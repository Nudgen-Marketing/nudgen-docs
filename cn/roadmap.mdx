以下是当前按优先级排序的规划功能。随着产品开发推进，时间安排和范围都可能调整。

## 1. 邮件模板

可复用的邮件模板，帮助你保存版式和文案，并在不同活动中重复使用。你可以先创建一次模板，在新建活动时直接选用，再根据需要做少量修改。我们也计划支持基于 Jinja 模板的**动态变量**，例如联系人姓名、产品名称以及其他按收件人或按活动变化的数据，让每次发送都更个性化。这会减少重复劳动，并帮助你保持统一的品牌表达。

## 2. 导出 CSV/Excel 报告

将活动和联系人数据导出为 CSV 或 Excel。你可以下载总览指标、收件人级活动数据或联系人列表，用于报表、备份或在其他工具中继续处理。

## 3. 活动 A/B 测试

对邮件主题或正文进行 A/B 测试。你可以把不同版本发送给不同的受众分组，再对比打开率、点击率和转化等表现。之后可以选择表现更好的版本，或将优胜版本推广到剩余的联系人。

## 4. 移动端支持

提供更适合手机和平板的使用体验，让你可以在移动设备上查看仪表板、审核活动并完成关键操作。我们会提供响应式布局和更适合触控的交互方式。

## 5. CRM API 集成

通过 API 将 Nudgen 与你的 CRM 连接起来：同步联系人、根据 CRM 事件触发活动，或把活动结果回传到 CRM。这样可以支持程序化工作流，并让多个系统之间的数据保持同步。

## 6. MCP 支持

支持 **MCP（Model Context Protocol）** 后，像 OpenClaw 这样的 LLM 以及其他兼容 MCP 的助手都能连接到 Nudgen 并控制活动。你可以在自己熟悉的 AI 环境中创建、启动、暂停或监控活动，并通过统一的工具接口完成操作。

## 7. 更多可组合的停止规则

我们会为停止规则加入更多条件，并允许多个条件组合使用。当设定条件满足时，活动可以自动暂停或停止，从而让你对送达率和名单健康有更细致的控制。

## 8. 基于数据洞察的 AI 活动建议

根据你的活动和联系人数据，给出发送时间、邮件主题或受众分组建议。这些建议会参考历史表现和互动情况，帮助你用更有依据的方式提升打开率、点击率和转化率。

## 9. 多语言支持（i18n）

在整个 Nudgen 生态中提供完整的国际化支持。我们正在为官网、文档站点和主应用仪表板引入多语言选项，为全球用户带来更顺畅一致的体验。
