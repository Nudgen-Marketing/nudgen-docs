## 快速安装

安装 Nudgen CLI 最简单的方法是使用我们的单行安装脚本。

### macOS 和 Linux

在您的终端中运行以下命令：

```bash
curl -fsSL https://raw.githubusercontent.com/Nudgen-Marketing/nudgen-cli/main/scripts/install.sh | bash
```

### 通过 Go 安装

如果您已安装 Go 1.22+，可以直接安装 CLI：

```bash
go install github.com/Nudgen-Marketing/nudgen-cli@latest
```

### 从源码构建

如果您已安装 Go 1.22+，可以从源代码构建：

```bash
git clone https://github.com/Nudgen-Marketing/nudgen-cli.git
cd nudgen-cli
make build
```

二进制文件将位于 `./bin/nudgen`。

## 初始设置

安装完成后，通过检查版本来验证安装：

```bash
nudgen version
```

### 身份认证

要开始使用 CLI，您需要使用个人访问令牌 (PAT) 进行身份认证。

1. 运行登录命令：
   ```bash
   nudgen login
   ```
2. 按照交互式提示输入您的令牌。
3. 验证您的身份：
   ```bash
   nudgen whoami
   ```

### 更新

要更新到最新版本，只需再次运行安装脚本或使用内置的更新命令（如果可用）：

```bash
nudgen update
```

## 安全性

Nudgen CLI 使用系统钥匙串安全地存储您的 PAT。我们绝不会在任何命令输出中记录或输出原始令牌。
