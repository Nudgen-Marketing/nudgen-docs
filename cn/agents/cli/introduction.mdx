## 概述

Nudgen CLI 是一款专为人类和 AI 智能体设计的强大工具。它提供了一种无缝的方式，可以直接从您的终端管理营销活动、联系人、品牌身份和推荐统计。

无论您是希望自动化营销工作流的开发人员，还是需要与 Nudgen 平台进行高质量交互的 AI 智能体，CLI 都是您的主要接口。

## 用途

CLI 扮演着多重角色：

1. **面向人类**：提供一种快速列出营销活动、导入联系人和查看推荐收益的方法，无需打开浏览器。
2. **面向 AI 智能体**：提供一个机器可解释的接口，以自动化复杂的营销推广和品牌管理。
3. **面向开发人员**：提供 Nudgen 生态系统内本地开发和仓库管理的辅助工具。

## 交互模式

CLI 遵循“交互优于 API”的原则。它使用标准的 RESTful API 调用与 Nudgen 后端通信，同时在本地安全地管理会话状态。

- **JSON 标准**：所有命令都支持 `--json`。这对于智能体之间的通信至关重要。
- **上下文感知**：诸如列出营销活动之类的操作会自动限定在您的活跃团队范围内。
