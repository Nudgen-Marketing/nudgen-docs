## 本节包含什么
这里汇总了 Nudgen 中最常见的问题，并按主题分类。如果你在这里没有找到答案，可以继续查看对应专题页面，或直接联系支持团队。

## 发送与送达率
| 问题 | 建议查看 |
| --- | --- |
| 邮件没有发出，或活动长时间卡在发送中 | [发送与送达率](/cn/troubleshooting/deliverability#电子邮件未发送) |
| 退信率或垃圾邮件投诉过高 | [发送与送达率](/cn/troubleshooting/deliverability#退信与投诉) |
| 测试邮件没有收到 | 检查垃圾邮件箱，并确认账号邮箱设置正确；也可以查看[创建活动](/cn/campaigns/create)中的说明 |
| 打开数或点击数没有更新 | [发送与送达率](/cn/troubleshooting/deliverability#追踪与指标) |
