## 概述

Nudgen CLI 是一款通用的强大工具，专为人类用户和 AI Agent 设计。它提供了一种无缝的方式，让您直接从终端管理营销活动、联系人、品牌标识和加盟统计数据。

无论您是希望自动化营销工作流程的开发人员，还是需要与 Nudgen 平台进行高质量交互的 AI Agent，CLI 都是您的主要界面。

## 目的

CLI 具备多种用途：

1. **面向人类用户**：快速列出营销活动、导入联系人及查看收益，无需打开浏览器。
2. **面向 AI Agent**：提供机器可解释的界面，以自动化复杂的营销推广和品牌管理。
3. **面向开发人员**：提供 Nudgen 生态系统内的本地开发和代码库管理助手。

## 交互模式

CLI 遵循“通过 API 进行交互”的原则。它使用标准 RESTful API 与 Nudgen 后端通信，并在本地安全管理会话状态。

- **JSON 标准**：所有命令均支持 `--json`，这对于 Agent 间的通信至关重要。
- **上下文感知**：列出营销活动等操作会自动限定在您的当前活跃团队范围内。
