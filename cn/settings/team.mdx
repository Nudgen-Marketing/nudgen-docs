## 本页可以帮助你做什么
在这里为 Nudgen 工作区添加团队成员、分配角色，并管理待处理或已过期的邀请。这样你可以更安全地与团队协作。

## 开始前
- 你需要 **Owner** 或 **Admin** 权限
- 准备好要邀请成员的邮箱地址

## 角色与权限
| 角色 | 可以做什么 |
| --- | --- |
| **Owner** | 拥有完整权限，可管理账单、团队和工作区设置。不能被移除，但可以转移所有权。 |
| **Admin** | 可以邀请和管理成员，编辑活动、联系人和品牌配置，但不能管理账单。 |
| **Member** | 可以创建和编辑活动、上传联系人、使用品牌配置。可以查看团队列表，但不能邀请成员或修改角色。 |

<Note>
只有 **Owner** 和 **Admin** 可以邀请新成员或修改角色。**Member** 只能查看团队列表，不能更改成员权限。
</Note>

## 邀请团队成员
1. 打开 **设置** → **团队**
2. 点击 **邀请成员**
3. 输入邮箱地址并选择角色，通常为 **Admin** 或 **Member**
4. 发送邀请，对方会收到带有接受链接的邮件

## 邀请状态
| 状态 | 含义 |
| --- | --- |
| **Pending** | 邀请已发出，收件人可以通过邮件中的链接接受。 |
| **Accepted** | 收件人已加入工作区。 |
| **Expired** | 邀请链接已过期，需要重新发送。 |
| **Cancelled** | 邀请在被接受前已被取消。 |

## 修改或移除成员
- **更改角色**：打开成员资料，选择新角色并保存
- **移除成员**：删除成员后，对方会立即失去工作区访问权限

## 接下来可以做什么
- 新成员加入后，可以[上传联系人](/cn/contacts/upload)、[创建活动](/cn/campaigns/create)，并使用共享的品牌配置
- **Owner** 仍然保留对账单和关键设置的控制权

## 常见问题
<AccordionGroup>
  <Accordion title="受邀者没有收到邀请邮件">
    让对方先检查垃圾邮件箱，并确认邮箱地址填写正确。必要时可以在团队页面重新发送邀请。
  </Accordion>
  <Accordion title="邀请已过期">
    邀请超过有效期后会失效。你可以用同一个邮箱重新创建邀请。
  </Accordion>
  <Accordion title="成员可以查看所有联系人吗？">
    可以。拥有访问权限的团队成员共享同一个工作区和联系人列表。你可以通过标签和活动来组织协作。
  </Accordion>
</AccordionGroup>
