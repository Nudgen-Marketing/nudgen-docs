## 本页可以帮助你做什么
了解 **仪表盘** 中的核心数据：工作区汇总指标、趋势图表，以及包含关键统计信息的活动列表。你可以用它快速判断发送健康度和整体表现。

## 开始前
- 你需要一个 Nudgen 账号，并且至少有一个工作区
- 如果工作区里已经有已发送或已排期的活动，图表和指标会更有参考价值

## 指标卡片
仪表盘会显示所选日期范围内的汇总指标：

| 指标 | 含义 |
| --- | --- |
| **发送尝试数** | 所有发送尝试的总数，包括 API 或网络失败的情况。 |
| **成功送达数** | 被收件方邮件服务器成功接受的邮件数量。 |
| **打开次数** | 邮件被打开的次数，通过追踪像素统计。 |
| **点击次数** | 邮件内链接被点击的总次数。 |
| **点击率** | `(点击次数 / 发送尝试数) × 100`。 |
| **垃圾邮件投诉** | 收件人将邮件标记为垃圾邮件的次数。 |
| **软退信** | 临时投递问题，例如邮箱已满、临时限流或服务器异常。 |
| **硬退信** | 永久投递失败，例如地址无效或域名被拒收。 |

这些数据可以帮助你判断送达情况和互动表现。较高的投诉率或退信率可能会影响发件信誉。

## 趋势图表
折线图展示所选日期范围内各项指标的变化趋势。你可以调整日期范围，用来观察打开率下降、退信激增，或活动上线后的表现变化。

## 活动列表
分页表格会列出活动名称、类型（单次或滴灌）、状态以及关键计数，如已发送、已送达、已打开、已点击、退信和投诉。点击某个活动后，可以进入详情页查看更细的收件人级数据和单活动趋势图。

## 日期范围
默认通常显示最近 7 天。你可以修改范围，聚焦某个具体时间段，或比较不同周的表现。所有统计和图表都会随之更新。

## 接下来可以看什么
