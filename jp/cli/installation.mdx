## クイックインストール

最も簡単なインストール方法は、ワンライナーのインストールスクリプトを使用することです。

### macOS & Linux

ターミナルで以下のコマンドを実行してください：

```bash
curl -fsSL https://raw.githubusercontent.com/Nudgen-Marketing/nudgen-cli/main/scripts/install.sh | bash
```

### ソースからビルド

Go 1.22+がインストールされている場合は、ソースからビルドできます：

```bash
git clone https://github.com/Nudgen-Marketing/nudgen-cli.git
cd nudgen-cli
make build
```

バイナリは`./bin/nudgen`に作成されます。

## 初期設定

インストール後、バージョンを確認して正常にインストールされたか確かめます：

```bash
nudgen version
```

### 認証

CLIの使用を開始するには、個人アクセスアクセストークン（PAT）で認証する必要があります。

1. ログインコマンドを実行します：
   ```bash
   nudgen login
   ```
2. インタラクティブなプロンプトに従ってトークンを入力します。
3. 同定を確認します：
   ```bash
   nudgen whoami
   ```

### アップデート

最新バージョンに更新するには、インストールスクリプトを再実行するか、内蔵のアップデートコマンドを使用します：

```bash
nudgen update
```

## セキュリティ

Nudgen CLIはシステムのキーチェーンを使用してPATを安全に保存します。コマンド出力に生のトークンを表示したりログに記録したりすることはありません。

次に、[コアコマンド](/jp/cli/commands)を探索しましょう。
