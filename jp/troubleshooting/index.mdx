## このセクションについて

よくある問題と回答を分野別にまとめています。ここで見つからない場合は、下の個別ガイドを確認するかサポートへ連絡してください。

## 送信と到達率

| 問題 | 確認先 |
| ---- | ------ |
| メールが送信されない / キャンペーンが止まった | [送信と到達率](/jp/troubleshooting/deliverability#emails-not-sending) |
| bounce や spam complaint が多い | [送信と到達率](/jp/troubleshooting/deliverability#bounces-and-complaints) |
| テストメールが届かない | [キャンペーン作成](/jp/campaigns/create) の troubleshooting |
| 開封やクリックが更新されない | [送信と到達率](/jp/troubleshooting/deliverability#tracking-and-metrics) |

## 連絡先とインポート

| 問題 | 確認先 |
| ---- | ------ |
| アップロード失敗や行のスキップ | [Contacts upload](/jp/contacts/upload#troubleshooting) |
| 連絡先上限を超えた | [Billing](/jp/billing/overview) |
| reach estimate が 0 | [Contacts overview](/jp/contacts/overview) |
| Reveal できない / マスキング | [Contacts overview](/jp/contacts/overview) |

## アクセスと権限

| 問題 | 確認先 |
| ---- | ------ |
| 招待が届かない | [アクセスと権限](/jp/troubleshooting/access#invitations) |
| 間違った workspace を見ている | [Workspaces and roles](/jp/concepts/workspaces-and-roles) |
| 招待やロール変更ができない | [Team settings](/jp/settings/team) |
| 削除後にアクセスできない | workspace owner に連絡 |

## Billing とサブスクリプション

| 問題 | 確認先 |
| ---- | ------ |
| 連絡先またはメール上限に達した | [請求トラブル](/jp/troubleshooting/billing#limits) |
| 支払い失敗 / カード拒否 | [請求トラブル](/jp/troubleshooting/billing#payments) |
| Trial 終了で送れない | [Billing overview](/jp/billing/overview) |
| プラン変更やキャンセル | [Billing overview](/jp/billing/overview) |
