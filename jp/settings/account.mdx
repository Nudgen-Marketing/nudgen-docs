## このページでできること

アカウント単位の設定を確認し、管理できます。プロフィール、チーム、署名、ブランド設定の入口として使えるページです。

## 始める前に

- Nudgen アカウント
- Owner または Admin 権限

## 設定一覧

ダッシュボードの **Settings** から次の設定にアクセスできます。

| セクション | できること | ドキュメント |
| ---------- | ---------- | ------------ |
| **Team** | メンバー招待、ロール設定、権限管理 | [Team settings](/jp/settings/team) |
| **Brand** | ブランド名、説明、reply-to、ロゴなどの設定 | [Brand overview](/jp/brand/overview) |
| **Signatures** | キャンペーンで使う署名の作成と編集 | [Signatures](/jp/settings/signatures) |

## ワークスペースとアカウント

あなたのアカウントは複数の workspace に所属できます。現在の workspace はサイドバー上部に表示され、team switcher で切り替えられます。連絡先、キャンペーン、ブランド設定、billing は active workspace 単位で管理されます。
