## このページでできること

Nudgen workspace にメンバーを追加し、ロールを割り当て、保留中または期限切れの招待を管理できます。

## 始める前に

- Owner または Admin 権限
- 招待したい相手のメールアドレス

## ロールと権限

| ロール | できること |
| ------ | ---------- |
| **Owner** | 全権限。Billing、チーム、workspace 設定を管理できます。削除は不可ですが所有権の移譲は可能です。 |
| **Admin** | メンバーの招待と管理、キャンペーン、連絡先、ブランド設定の編集。Billing 権限はありません。 |
| **Member** | キャンペーン作成と編集、連絡先のアップロード、ブランド設定や署名の利用。チーム一覧は見られますが変更はできません。 |

<Note>
新しいメンバーの招待やロール変更ができるのは Owner と Admin だけです。
</Note>

## メンバーを招待する

1. **Settings** → **Team** に移動します。
2. **Invite member** をクリックします。
3. メールアドレスを入力し、ロールを選びます。
4. 送信すると、相手に参加用リンク付きメールが届きます。

## 招待ステータス

| ステータス | 意味 |
| ---------- | ---- |
| **Pending** | 招待送信済み。メール内リンクから参加できます。 |
| **Accepted** | 招待を受けて参加済みです。 |
| **Expired** | 招待リンクの期限切れです。再送が必要です。 |
| **Cancelled** | 承認前に招待が取り消されました。 |

## メンバーの変更や削除

- **Change role:** ロールを変更して保存します
- **Remove member:** 削除を確定すると即時にアクセスを失います

## 次のステップ

- 新しいメンバーは [連絡先のアップロード](/jp/contacts/upload) や [キャンペーン作成](/jp/campaigns/create) を行えます。
- Billing と重要設定の管理は引き続き Owner が担います。
