## このページでできること

ブランドのグローバルアイデンティティを設定します。この情報はメールの「From」ヘッダーに使用され、AIがより正確な生成を行うためのコンテキストを提供し、キャンペーンコピーのデフォルト言語を設定します。

## 開始前の準備

- Nudgenアカウント
- オーナーまたは管理者ロール
- ブランドアセット（名前、ロゴ、送信者メール）

## ブランド設定

ダッシュボードの **設定** → **ブランド** から以下のオプションにアクセスできます：

| オプション           | 説明                                                                 |
| -------------------- | --------------------------------------------------------------------------- |
| **ブランド名**       | 会社名または製品名（例：「Nudgen」）。AIのコンテキストに使用されます。  |
| **ブランドメール**   | キャンペーンのデフォルト返信先メールアドレス（例：`support@yourbrand.com`）。 |
| **ブランドの説明**   | ブランドの簡単な概要。AIがより良いコピーを書くのに役立ちます。    |
| **メール言語**       | AIが件名や本文を生成する際に使用するデフォルトの言語。              |
| **ロゴ**             | ブランドのロゴ（PNGまたはJPEG、最大5MB）。メールテンプレートに使用されます。 |

## サポートされているメール言語

Nudgenは、AIによって生成されるメールコピーについて幅広い言語をサポートしています。デフォルトは **設定** → **ブランド** で設定できますが、キャンペーンごとに変更することも可能です。
