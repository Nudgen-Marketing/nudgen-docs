## 仕組み

受動所得へのシンプルな3つのステップ。

1. **独自のリンクを取得** — パートナーとして登録し、30日間のすべてのクリックとコンバージョンを追跡する個人用リファラルリンクを受け取ります。
2. **自由にプロモーション** — リンクをどこでも共有できます。有料広告、SNS、ブログ、メールなどが可能です。マーケティングの自由度は100%です。
3. **生涯受動所得** — 紹介したユーザーがサブスクリプションを更新するたびに、10%の継続報酬を獲得できます。上限なしで一生涯続きます。

## 報酬体系

すべての有料プランにおいて報酬は継続的です。紹介顧客が増えるほど、収益も成長します。

| :--- | :--- |

| **クッキー有効期間** | 30日間 |
| **最低支払い額** | $50 USD |
| **支払い方法** | PayPal |
| **承認プロセス** | 即時 / 手動承認不要 |
