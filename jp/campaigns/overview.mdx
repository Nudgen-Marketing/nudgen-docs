## このページでできること

2 種類のキャンペーンと、それぞれのステータスの意味を理解できます。どの形式を選ぶべきか、そしてキャンペーン一覧をどう読み取るかがわかります。

## 始める前に

- Nudgen に連絡先がアップロードされていること
- プリセットトーンまたはカスタムプロンプトを含むブランド設定

## キャンペーンの種類

### 単発送信キャンペーン

選択したオーディエンスに 1 回だけ送るメールです。次のような場面で使います。

- 新機能やイベントのお知らせ
- 一度きりの再エンゲージメント配信
- 長いシーケンスを作る前のメッセージ検証

オーディエンスを選び、AI が内容を生成し、プレビューしてから送信またはスケジュールできます。

### ドリップシーケンス

Day 1、Day 3、Day 7 のように、メール間に間隔を置いて送る複数ステップのキャンペーンです。次のような用途に向いています。

- 非アクティブなユーザーを数日かけて呼び戻す
- 開封しなかった人やクリックしなかった人へのフォローアップ
- Duolingo のようなリテンションフローを構築する

各ステップにはそれぞれ目標、件名、AI 生成の本文があります。連絡先は設定された遅延が経過した後に次のステップへ進みます。**auto-stop rules** を設定すれば、たとえばクリックや開封があった受信者への後続ステップを止められます。

**オーディエンスの種類:** キャンペーン作成時に **All contacts**、**By tags**、**Manual** から送信先を選べます。ウィザードには配信前に想定リーチ数が表示されます。

## キャンペーンステータス

| ステータス | 意味 |
| ---------- | ---- |
| **Draft** | まだ送信されていません。目標、オーディエンス、本文、スケジュールを編集できます。 |
| **Scheduled** | 送信時刻が設定されています。指定時刻になると送信されます。 |
| **Active** / **Sending** | メールが送信中です。進行状況はダッシュボードに表示されます。 |
| **Sent** | すべてのメール送信が完了しています。本文はロックされますが、分析は確認できます。 |
| **Stopped** | 手動で停止された状態です。一部の連絡先にはすでに送信済みの可能性があり、今後のドリップステップはキャンセルされます。 |

<Note>
キャンペーンを一度送信すると、内容を編集したり再送したりはできません。別のメッセージを送りたい場合は新しいキャンペーンを作成してください。
</Note>
