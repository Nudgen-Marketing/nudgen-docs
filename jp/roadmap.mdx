優先順位に基づいた計画中の機能です。開発状況により、日程や範囲は変更される可能性があります。

## 1. キャンペーンの A/B テスト

メールの件名やコンテンツで A/B テストを実行します。オーディエンスのセグメントにバリアントを送信し、パフォーマンス（開封率、クリック率、コンバージョン率）を比較します。結果を基に勝者を選択するか、最適なバリアントをリストの残りのメンバーに展開します。

## 2. 複数の停止ルールの組み合わせサポート

停止ルールの条件を増やし、それらを組み合わせることができるようになります。組み合わせた条件が満たされると、キャンペーンは自動的に一時停止または停止し、よりきめ細かな制御と、到達率およびリストの健全性の向上を実現します。

## 3. 行動追跡 SDK と早期離脱防止

製品に埋め込む軽量な SDK で、個々の顧客の行動（ページ訪問、機能の使用、非アクティブなパターン、エンゲージメント信号）を連絡先レベルで追跡します。事後に集計トレンドを表示する従来の分析ツールとは異なり、Nudgen の SDK はアクションのために構築されています。連絡先ごとの早期離脱の兆候をリアルタイムで検出し、離脱リスクをスコアリングして、顧客が離れる前に再エンゲージメントキャンペーンを自動的に提案します。顧客が去っていくのをただ眺めるのではなく、まだ間に合ううちに対処しましょう。
