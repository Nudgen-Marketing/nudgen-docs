---

現在予定している機能を優先度順にまとめています。日程や範囲は開発の進行にあわせて変更される場合があります。

## 1. メールテンプレート

レイアウトや本文を保存し、複数のキャンペーンで再利用できるメールテンプレートです。一度テンプレートを作成すれば、キャンペーン作成時に選択して必要な部分だけを調整できます。さらに、連絡先名や製品名など、受信者ごと・キャンペーンごとに変わるデータを差し込める **Jinja テンプレートによる動的変数** にも対応予定です。これにより、繰り返し作業を減らしつつ、ブランドの一貫性も保てます。

## 2. キャンペーンの A/B テスト

件名やメール本文の A/B テストを実施できるようにします。異なるバリエーションを受信者グループごとに送り分け、開封率、クリック率、コンバージョンなどを比較できます。その結果から勝ちパターンを選んだり、残りの受信者に最適なバージョンを展開したりできます。

## 3. CRM API 連携

API を通じて Nudgen を CRM と連携できるようにします。連絡先の同期、CRM イベントを起点にしたキャンペーン実行、キャンペーン結果の CRM への返却などが可能になり、より自動化されたワークフローを構築できます。

## 4. より柔軟な停止ルール

停止ルールの条件を増やし、複数条件の組み合わせにも対応します。設定した条件を満たしたときにキャンペーンを自動で停止または一時停止できるため、より細かい制御ができ、到達性やリストの健全性も守りやすくなります。

## 5. データに基づく AI キャンペーン提案

キャンペーンや連絡先のデータに基づいて、最適な送信時間、件名、セグメントなどを提案します。過去の成果やエンゲージメントを参考にした提案により、勘に頼らず開封率、クリック率、コンバージョンの改善を目指せます。
