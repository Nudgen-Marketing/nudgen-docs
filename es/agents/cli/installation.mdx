## Instalación Rápida

La forma más sencilla de instalar la CLI de Nudgen es utilizando nuestro script de instalación de una sola línea.

### macOS y Linux

Ejecute el siguiente comando en su terminal:

```bash
curl -fsSL https://raw.githubusercontent.com/Nudgen-Marketing/nudgen-cli/main/scripts/install.sh | bash
```

### Mediante Go

Si tiene instalado Go 1.22+, puede instalar la CLI directamente:

```bash
go install github.com/Nudgen-Marketing/nudgen-cli@latest
```

### Desde el Código Fuente

Si tiene instalado Go 1.22+, puede compilar desde el código fuente:

```bash
git clone https://github.com/Nudgen-Marketing/nudgen-cli.git
cd nudgen-cli
make build
```

El binario estará disponible en `./bin/nudgen`.

## Configuración Inicial

Una vez instalado, verifique la instalación comprobando la versión:

```bash
nudgen version
```

### Autenticación

To start using the CLI, you need to authenticate with a Personal Access Token (PAT).
Para empezar a usar la CLI, necesita autenticarse con un Token de Acceso Personal (PAT).

1. Ejecute el comando de inicio de sesión:
   ```bash
   nudgen login
   ```
2. Siga las instrucciones interactivas para introducir su token.
3. Verifique su identidad:
   ```bash
   nudgen whoami
   ```

### Actualización

Para actualizar a la última versión, simplemente ejecute de nuevo el script de instalación o utilice el comando de actualización integrado (si está disponible):

```bash
nudgen update
```

## Seguridad

La CLI de Nudgen utiliza el llavero del sistema para almacenar su PAT de forma segura. Nunca registramos ni mostramos tokens sin procesar en la salida de ningún comando.
