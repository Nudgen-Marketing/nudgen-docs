## Resumen

La CLI de Nudgen es una herramienta potente diseñada tanto para humanos como para agentes de IA. Proporciona una forma fluida de gestionar campañas, contactos, identidades de marca y estadísticas de referidos directamente desde su terminal.

Ya sea un desarrollador que busca automatizar sus flujos de trabajo de marketing o un agente de IA que necesita una interacción de alta calidad con la plataforma Nudgen, la CLI es su interfaz principal.

## Propósito

La CLI cumple múltiples funciones:

1. **Para Humanos**: Proporciona una forma rápida de listar campañas, importar contactos y consultar las ganancias de referidos sin abrir el navegador web.
2. **Para Agentes de IA**: Ofrece una interfaz interpretable por máquinas para automatizar la comunicación de marketing compleja y la gestión de marca.
3. **Para Desarrolladores**: Ayudas para el desarrollo local y la gestión de repositorios dentro del ecosistema de Nudgen.

## Patrones de Interacción

La CLI sigue el principio de "Interacción sobre API". Utiliza llamadas API RESTful estándar para comunicarse con el backend de Nudgen mientras gestiona de forma segura el estado de la sesión localmente.

- **Estándar JSON**: Todos los comandos admiten `--json`. Esto es fundamental para la comunicación entre agentes.
- **Conciencia del Contexto**: Las operaciones como el listado de campañas se limitan automáticamente a su equipo activo.
