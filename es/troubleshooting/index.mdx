## Qué cubre esta sección
Problemas comunes y respuestas agrupadas por área. Si aquí no encuentras tu caso, consulta las guías específicas que aparecen abajo o contacta con soporte.

## Envío y entregabilidad
| Problema | Dónde mirar |
| --- | --- |
| Los correos no se envían o la campaña se queda atascada | [Envío y entregabilidad](/es/troubleshooting/deliverability#correos-que-no-se-envían) |
| Rebotes altos o muchas quejas de spam | [Envío y entregabilidad](/es/troubleshooting/deliverability#rebotes-y-quejas) |
| El correo de prueba no llegó | Revisa spam y confirma que el correo de la cuenta sea correcto. Consulta la ayuda de [Crear campañas](/es/campaigns/create). |
| Las aperturas o los clics no se actualizan | [Envío y entregabilidad](/es/troubleshooting/deliverability#seguimiento-y-métricas) |

</Card>

## Contactos e importaciones
| Problema | Dónde mirar |
| --- | --- |
| La carga falló o se omitieron filas | [Subir contactos](/es/contacts/upload#solución-de-problemas): revisa formato CSV, columna de correo y límite del plan. |
| Se superó el límite de contactos | Mejora tu plan o elimina contactos. [Facturación](/es/billing/overview). |
| La estimación de alcance es cero | Añade o importa contactos y asegúrate de que estén activos. [Resumen de contactos](/es/contacts/overview). |
| No puedo mostrar un contacto o sigue enmascarado | Necesitas acceso al workspace; la visualización se hace por sesión y vuelve a ocultarse al actualizar. [Resumen de contactos](/es/contacts/overview). |

## Acceso y permisos
| Problema | Dónde mirar |
| --- | --- |
| No llegó la invitación | [Acceso y permisos](/es/troubleshooting/access#invitaciones): revisa spam, reenvía y confirma el correo correcto. |
| Estoy en el workspace equivocado o no veo datos | Usa el selector de equipo de la barra lateral. [Espacios de trabajo y roles](/es/concepts/workspaces-and-roles). |
| No puedo invitar ni cambiar roles | Solo pueden hacerlo owners y admins. [Configuración del equipo](/es/settings/team). |
| Perdí acceso después de que me quitaran del equipo | Contacta con el owner del workspace. La facturación es solo para owners. |

## Facturación y suscripciones
| Problema | Dónde mirar |
| --- | --- |
| Alcancé el límite de contactos o de correos | [Facturación y suscripciones](/es/troubleshooting/billing#límites): mejora el plan o espera al reinicio. |
| El pago falló o la tarjeta fue rechazada | [Facturación y suscripciones](/es/troubleshooting/billing#pagos): actualiza la tarjeta en el portal. |
| La prueba terminó y no puedo enviar | Pásate a un plan de pago. [Resumen de facturación](/es/billing/overview). |
| Quiero cambiar o cancelar el plan | Usa el portal de facturación, desde **Gestionar suscripción**. [Resumen de facturación](/es/billing/overview). |

<Card title="Facturación y suscripciones" icon="credit-card" href="/es/troubleshooting/billing">

</Card>

## Campañas y contenido
| Problema | Dónde mirar |
| --- | --- |
| El contenido de IA no coincide con mi tono | Ajusta la configuración de marca con prompts personalizados y ejemplos. [Resumen de marca](/es/brand/overview). |
| No puedo editar una campaña | Solo se pueden editar borradores y campañas programadas. [Gestionar campañas](/es/campaigns/manage). |
| Cómo detengo una campaña | Usa **Detener campaña** en la lista o en la página de detalle. [Gestionar campañas](/es/campaigns/manage). |
| La firma falta o es incorrecta | Selecciona la firma en el asistente de campaña y revisa la vista previa. [Firmas](/es/settings/signatures). |
