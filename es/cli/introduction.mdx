## Resumen

La CLI de Nudgen es una herramienta potente diseñada tanto para humanos como para agentes de IA. Ofrece una forma fluida de gestionar campañas, contactos, identidades de marca y estadísticas de referidos directamente desde su terminal.

Ya sea que sea un desarrollador que busca automatizar sus flujos de trabajo de marketing o un agente de IA que necesita una interacción de alta calidad con la plataforma Nudgen, la CLI es su interfaz principal.

## Propósito

La CLI cumple múltiples funciones:

1. **Para Humanos**: Permite listar campañas rápidamente, importar contactos y consultar ganancias sin necesidad de abrir el navegador.
2. **Para Agentes de IA**: Ofrece una interfaz interpretable por máquinas para automatizar el marketing y la gestión de marca de forma compleja.
3. **Para Desarrolladores**: Ayudas para el desarrollo local y la gestión del repositorio dentro del ecosistema Nudgen.

## Patrones de Interacción

La CLI sigue el principio de "Interacción sobre API". Utiliza llamadas API RESTful estándar para comunicarse con el backend de Nudgen mientras gestiona el estado de la sesión localmente de forma segura.

- **Estándar JSON**: Todos los comandos admiten `--json`, lo cual es fundamental para la comunicación entre agentes.
- **Conciencia del Contexto**: Las operaciones se limitan automáticamente a su equipo activo.
