## Apa itu Ruang Kerja?

**Ruang kerja** (tim) adalah wadah untuk semua data bersama di Nudgen: kontak, kampanye, konfigurasi merek, tanda tangan, dan penagihan. Setiap ruang kerja memiliki langganan dan batas penggunaan sendiri. Akun Anda dapat terhubung ke beberapa ruang kerja; Anda dapat beralih di antaranya dengan **pemilih tim** di bilah sisi.

## Peran dan Izin

| Peran | Kemampuan |
| ---- | ------------- |
| **Owner** | Akses penuh. Mengelola penagihan, tim, dan pengaturan ruang kerja. Tidak dapat dihapus; dapat mentransfer kepemilikan. |
| **Admin** | Mengundang dan mengelola anggota, mengedit kampanye, kontak, dan merek. Tidak ada akses penagihan. |
| **Member** | Membuat dan mengedit kampanye, mengunggah kontak, menggunakan konfigurasi merek dan tanda tangan. Dapat melihat daftar tim tetapi tidak dapat mengundang atau mengubah peran. |

Hanya pemilik (owner) dan admin yang dapat mengundang anggota baru atau mengubah peran. Penagihan (peningkatan paket, metode pembayaran, portal) dibatasi hanya untuk pemilik.

## Undangan

Admin dan pemilik mengirim **undangan** melalui email. Orang yang diundang akan menerima tautan untuk menerima dan bergabung ke dalam ruang kerja. Undangan dapat berstatus tertunda, diterima, kedaluwarsa, atau dibatalkan. Kirim ulang dari **Pengaturan** → **Tim** jika diperlukan.

## Ruang Kerja Aktif

Ruang kerja yang ditampilkan di header bilah sisi adalah ruang kerja **aktif** Anda. Semua halaman (Dasbor, Kontak, Kampanye, Merek, Penagihan, Pengaturan) menampilkan data untuk ruang kerja tersebut. Alihkan ruang kerja untuk bekerja di tim yang berbeda; profil Anda (nama, email, avatar) tetap sama di semua ruang kerja.
