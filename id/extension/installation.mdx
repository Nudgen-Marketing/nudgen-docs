---

Lead Email Finder kompatibel dengan **Google Chrome, Microsoft Edge**, semua browser berbasis Chromium, dan **Mozilla Firefox**.

## Browser yang Didukung

Pilih browser Anda di bawah ini untuk mengakses toko ekstensi resmi:

<CardGroup cols={3}>
  <Card title="Google Chrome" icon="chrome" href="https://chromewebstore.google.com/">
    Tambahkan ke Chrome
  </Card>
  <Card title="Microsoft Edge" icon="edge" href="https://microsoftedge.microsoft.com/addons/">
    Tambahkan ke Edge
  </Card>
  <Card title="Mozilla Firefox" icon="firefox-browser" href="https://addons.mozilla.org/">
    Tambahkan ke Firefox
  </Card>
</CardGroup>

<Note>
Pengguna Brave, Opera, dan browser berbasis Chromium lainnya dapat menginstal ekstensi langsung dari Chrome Web Store.
</Note>

## Panduan Penyiapan

1. **Kunjungi Toko**: Klik tautan di atas untuk browser Anda.
2. **Tambahkan Ekstensi**: Klik "Tambahkan ke Chrome" (atau setara) dan konfirmasikan izin.
3. **Sematkan ke Toolbar**: Untuk akses mudah, klik ikon potongan teka-teki di browser Anda dan sematkan **Lead Email Finder**.
4. **Mulai Mencari**: Buka profil atau situs web apa pun dan klik ikon ekstensi untuk memulai.

<Note>
Anda dapat menggunakan ekstensi secara gratis dengan batas pemindaian harian. Untuk pemindaian tak terbatas, tingkatkan ke [paket Pro](https://nudgen.net/pricing).
</Note>
