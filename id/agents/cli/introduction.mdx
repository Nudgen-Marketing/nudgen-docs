## Ikhtisar

Nudgen CLI adalah alat canggih yang dirancang untuk manusia dan agen AI. Alat ini menyediakan cara yang mulus untuk mengelola kampanye, kontak, identitas merek, dan statistik referensi langsung dari terminal Anda.

Baik Anda seorang pengembang yang ingin mengotomatiskan alur kerja pemasaran Anda atau agen AI yang membutuhkan interaksi berkualitas tinggi dengan platform Nudgen, CLI adalah antarmuka utama Anda.

## Tujuan

CLI melayani berbagai peran:

1. **Untuk Manusia**: Menyediakan cara cepat untuk mencantumkan kampanye, mengimpor kontak, dan memeriksa penghasilan referensi tanpa membuka browser web.
2. **Untuk Agen AI**: Menawarkan antarmuka yang dapat diinterpretasikan mesin untuk mengotomatiskan penjangkauan pemasaran dan manajemen merek yang kompleks.
3. **Untuk Pengembang**: Pembantu untuk pengembangan lokal dan manajemen repositori dalam ekosistem Nudgen.

## Pola Interaksi

CLI mengikuti prinsip "Interaction over API". Ia menggunakan panggilan RESTful API standar untuk berkomunikasi dengan backend Nudgen sambil mengelola status sesi secara lokal dengan aman.

- **Standar JSON**: Semua perintah mendukung `--json`. Ini sangat penting untuk komunikasi antar agen.
- **Sadar Konteks**: Operasi seperti mencantumkan kampanye secara otomatis dicakupkan ke tim aktif Anda.
