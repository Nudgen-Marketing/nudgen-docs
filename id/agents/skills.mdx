## Tambahkan kemampuan ke agen Anda

Nudgen Skills adalah instruksi dan skrip modular yang dirancang untuk agen coding seperti Claude Code, Codex, dan Cursor. Mereka memperluas pemahaman agen Anda tentang alur kerja pemasaran dan pola integrasi Nudgen.

## Keterampilan yang Tersedia

### CLI Skill
Instal dan autentikasi Nudgen CLI, kelola tim, kontak, kampanye, identitas merek, dan pelacakan referensi.

```bash
npx skills add https://github.com/Nudgen-Marketing/skills --global --skill cli
```

**Detail**: Instal, autentikasi melalui PAT, dan kelola kontak, kampanye, identitas merek, dan data referensi dari terminal. Mendukung perpindahan konteks multi-tim dan output `--json` untuk pembuatan skrip.

---

### API Skill
CRUD Kontak, kampanye, tim, konfigurasi merek, statistik, data afiliasi, pola SDK, batas rate limit, dan kunci idempotensi.

```bash
npx skills add https://github.com/Nudgen-Marketing/skills --global --skill api
```

**Detail**: Mencakup kontak, kampanye, templat, pengaturan merek, manajemen tim, pola SDK vs REST API, batas rate limit, kunci idempotensi, dan penanganan kesalahan. Menggunakan `Authorization: Bearer <PAT>` untuk autentikasi.

---

### Email Best Practices Skill
Keterampilan email serbaguna. Mengaudit deliverability, struktur, dan pola siklus hidup. Bekerja dengan atau tanpa Nudgen.

```bash
npx skills add https://github.com/Nudgen-Marketing/skills --global --skill email-sending-best-practices
```

**Detail**: Memvalidasi autentikasi pengirim (SPF, DKIM, DMARC), memeriksa pemisahan antara email transaksional vs pemasaran, meninjau penanganan berhenti berlangganan, menandai tahap siklus hidup yang hilang, dan memberi skor konten untuk pemicu spam.
