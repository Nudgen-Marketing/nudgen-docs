## Ringkasan

Kenali perintah inti yang tersedia di Nudgen CLI. Sebagian besar perintah dikelompokkan berdasarkan jenis sumber daya (kampanye, kontak, tim, dll.).

### Autentikasi & Identitas

| Perintah | Deskripsi |
| :--- | :--- |
| `nudgen login` | Autentikasi dengan Personal Access Token (PAT) Anda. |
| `nudgen whoami` | Tampilkan pemilik token saat ini dan informasi pengguna. |
| `nudgen logout` | Hapus kredensial lokal. |

### Manajemen Tim

| Perintah | Deskripsi |
| :--- | :--- |
| `nudgen teams list` | Daftar semua tim tempat Anda bergabung. |
| `nudgen teams current` | Tampilkan konteks tim yang aktif saat ini. |
| `nudgen teams switch <id>` | Beralih ke tim tertentu. |

### Kampanye

Kelola upaya pemasaran Anda dalam tim yang aktif.

| Perintah | Deskripsi |
| :--- | :--- |
| `nudgen campaigns list` | Daftar kampanye aktif di tim saat ini. |
| `nudgen campaigns create` | Wisaya pembuatan interaktif. |
| `nudgen campaigns update <id>` | Ubah pengaturan kampanye yang ada. |
| `nudgen campaigns delete <id>` | Hapus catatan kampanye. |

### Kontak (Manajemen Prospek)

| Perintah | Deskripsi |
| :--- | :--- |
| `nudgen contacts list` | Daftar semua prospek di tim saat ini. |
| `nudgen contacts create` | Tambahkan catatan prospek secara manual. |
| `nudgen contacts import` | Pemeta CSV interaktif untuk asupan massal. |
| `nudgen contacts update <id>` | Edit detail prospek. |
| `nudgen contacts delete <id>` | Hapus catatan kontak. |

### Identitas Merek (Brand)

| Perintah | Deskripsi |
| :--- | :--- |
| `nudgen brand list` | Daftar semua suara merek tim. |
| `nudgen brand create` | Wisaya untuk menginisialisasi suara merek baru. |
| `nudgen brand update <id>` | Edit konfigurasi merek. |
| `nudgen brand delete <id>` | Hapus identitas merek. |

### Referensi & Penghasilan

| Perintah | Deskripsi |
| :--- | :--- |
| `nudgen referral stats` | Lihat ringkasan kinerja tingkat tinggi dari referensi. |
| `nudgen referral link` | Dapatkan tautan rujukan unik Anda. |
| `nudgen referral activity` | Lihat riwayat klik terbaru dan pendaftaran. |
| `nudgen referral referrals` | Daftar tim yang Anda rujuk dan status mereka. |
| `nudgen referral payouts` | Lihat riwayat permintaan pembayaran dan statusnya. |

Selanjutnya, lihat cara menggunakan perintah ini untuk [Agen AI](/id/cli/agentic-usage).
