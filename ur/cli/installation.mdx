## فوری تنصیب

Nudgen CLI کو انسٹال کرنے کا سب سے آسان طریقہ ہماری ون لائنر انسٹالیشن اسکرپٹ کا استعمال ہے۔

### macOS اور Linux

اپنے ٹرمینل میں درج ذیل کمانڈ چلائیں:

```bash
curl -fsSL https://raw.githubusercontent.com/Nudgen-Marketing/nudgen-cli/main/scripts/install.sh | bash
```

### سورس سے بلڈ کریں

اگر آپ کے پاس Go 1.22+ انسٹال ہے، تو آپ اسے سورس سے خود بلڈ کر سکتے ہیں:

```bash
git clone https://github.com/Nudgen-Marketing/nudgen-cli.git
cd nudgen-cli
make build
```

بائنری فائل `./bin/nudgen` میں تیار ہو جائے گی۔

## ابتدائی ترتیب

انسٹالیشن کے بعد، ورژن چیک کر کے تصدیق کریں کہ سب کچھ درست طریقے سے انسٹال ہوا ہے:

```bash
nudgen version
```

### لاگ ان

CLI کا استعمال شروع کرنے کے لیے آپ کو پرسنل ایکسیس ٹوکن (PAT) کے ذریعے لاگ ان کرنا ہوگا۔

1. لاگ ان کمانڈ چلائیں:
   ```bash
   nudgen login
   ```
2. اسکرین پر آنے والی ہدایات کے مطابق اپنا ٹوکن درج کریں۔
3. اپنی شناخت کی تصدیق کریں:
   ```bash
   nudgen whoami
   ```

### اپ ڈیٹ

تازہ ترین ورژن پر اپ ڈیٹ کرنے کے لیے، صرف انسٹالیشن اسکرپٹ دوبارہ چلائیں یا بلٹ ان اپ ڈیٹ کمانڈ استعمال کریں:

```bash
nudgen update
```

## سیکیورٹی

Nudgen CLI آپ کے ٹوکن کو محفوظ طریقے سے سٹور کرنے کے لیے سسٹم کی چین (Keychain) کا استعمال کرتا ہے۔ ہم کبھی بھی کمانڈ آؤٹ پٹ یا لاگز میں آپ کا ٹوکن ظاہر نہیں کرتے۔

اگلے مرحلے میں، [بنیادی کمانڈز](/ur/cli/commands) دیکھیں۔
