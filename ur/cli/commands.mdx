## جائزہ

Nudgen CLI میں دستیاب اہم کمانڈز کے بارے میں جانیں۔ زیادہ تر کمانڈز کو وسائل کی قسم (مہمات، رابطے، ٹیمیں وغیرہ) کے مطابق گروپ کیا گیا ہے۔

### لاگ ان اور شناخت

| کمانڈ | تفصیل |
| :--- | :--- |
| `nudgen login` | اپنے پرسنل ایکسیس ٹوکن (PAT) کے ساتھ لاگ ان کریں۔ |
| `nudgen whoami` | موجودہ ٹوکن کے مالک اور صارف کی معلومات دیکھیں۔ |
| `nudgen logout` | لوکل لاگ ان تفصیلات کو ختم کریں۔ |

### ٹیم مینیجمنٹ

| کمانڈ | تفصیل |
| :--- | :--- |
| `nudgen teams list` | ان تمام ٹیموں کی فہرست دیکھیں جن کا آپ حصہ ہیں۔ |
| `nudgen teams current` | موجودہ فعال ٹیم کا تناظر دیکھیں۔ |
| `nudgen teams switch <id>` | کسی مخصوص ٹیم پر سوئچ کریں۔ |

### مہمات (Campaigns)

اپنی فعال ٹیم کے اندر مارکیٹنگ کی کوششوں کو مینیج کریں۔

| کمانڈ | تفصیل |
| :--- | :--- |
| `nudgen campaigns list` | موجودہ ٹیم میں فعال مہمات کی فہرست دیکھیں۔ |
| `nudgen campaigns create` | انٹرایکٹو بنانے کا طریقہ۔ |
| `nudgen campaigns update <id>` | موجودہ مہم کی ترتیبات تبدیل کریں۔ |
| `nudgen campaigns delete <id>` | کسی مہم کا ریکارڈ حذف کریں۔ |

### رابطے (Lead Management)

| کمانڈ | تفصیل |
| :--- | :--- |
| `nudgen contacts list` | موجودہ ٹیم میں تمام لیڈز کی فہرست دیکھیں۔ |
| `nudgen contacts create` | دستی طور پر نیا لیڈ ریکارڈ شامل کریں۔ |
| `nudgen contacts import` | بڑی تعداد میں امپورٹ کرنے کے لیے انٹرایکٹو CSV میپر۔ |
| `nudgen contacts update <id>` | لیڈ کی تفصیلات ایڈٹ کریں۔ |
| `nudgen contacts delete <id>` | رابطے کا ریکارڈ حذف کریں۔ |

### برانڈ کی شناخت (Brand)

| کمانڈ | تفصیل |
| :--- | :--- |
| `nudgen brand list` | ٹیم کی تمام برانڈ آوازوں (Tone) کی فہرست دیکھیں۔ |
| `nudgen brand create` | نئی برانڈ آواز شروع کرنے کا طریقہ۔ |
| `nudgen brand update <id>` | برانڈ کی ترتیب کو ایڈٹ کریں۔ |
| `nudgen brand delete <id>` | برانڈ کی شناخت حذف کریں۔ |

### ایفی لیٹ اور آمدنی

| کمانڈ | تفصیل |
| :--- | :--- |
| `nudgen referral stats` | ایفی لیٹ کی کارکردگی کا خلاصہ دیکھیں۔ |
| `nudgen referral link` | اپنا منفرد ریفرل لنک حاصل کریں۔ |
| `nudgen referral activity` | حالیہ کلکس اور سائن اپس کی تاریخ دیکھیں۔ |
| `nudgen referral referrals` | اپنی ریفر کردہ ٹیموں اور ان کی صورتحال کی فہرست دیکھیں۔ |
| `nudgen referral payouts` | ادائیگی کی درخواستوں کی تاریخ اور صورتحال دیکھیں۔ |

اگلے مرحلے میں، دیکھیں کہ ان کمانڈز کو [AI ایجنٹوں](/ur/cli/agentic-usage) کے لیے کیسے استعمال کیا جائے۔
