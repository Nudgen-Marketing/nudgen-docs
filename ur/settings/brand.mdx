## یہ صفحہ آپ کی کیا مدد کرتا ہے

اپنے برانڈ کی عالمی شناخت ترتیب دیں۔ یہ معلومات آپ کی ای میلز میں "From" ہیڈرز کے لیے استعمال کی جاتی ہیں، ای میلز کی زیادہ درست جنریشن کے لیے AI کو سیاق و سباق فراہم کرتی ہیں، اور آپ کی مہم کے مواد کے لیے پہلے سے طے شدہ زبان متعین کرتی ہیں۔

## شروع کرنے سے پہلے

- ایک Nudgen اکاؤنٹ
- مالک (Owner) یا ایڈمن کا رول
- آپ کے برانڈ کے اثاثے (نام، لوگو اور بھیجنے والے کی ای میل)

## برانڈ کی شناخت کی ترتیبات

درج ذیل اختیارات تک رسائی کے لیے ڈیش بورڈ میں **Settings** ← **Brand** پر جائیں:

| آپشن | تفصیل |
| :--- | :--- |
| **Brand name** | آپ کی کمپنی یا پروڈکٹ کا نام (مثلاً "Nudgen")۔ AI کے سیاق و سباق میں استعمال ہوتا ہے۔ |
| **Brand email** | آپ کی مہمات کے لیے پہلے سے طے شدہ جواب دہ (reply-to) ای میل ایڈریس (مثلاً `support@yourbrand.com`)۔ |
| **Brand description** | آپ کا برانڈ کیا کرتا ہے اس کا مختصر خلاصہ۔ AI کو بہتر مواد لکھنے میں مدد دیتا ہے۔ |
| **Email language** | وہ پہلے سے طے شدہ زبان جس میں AI سبجیکٹ لائنز اور ای میل کا متن تیار کرتا ہے۔ |
| **Logo** | آپ کے برانڈ کا لوگو (PNG یا JPEG، زیادہ سے زیادہ 5MB)۔ ای میل ٹیمپلیٹس اور پری ویو کے لیے استعمال ہوتا ہے۔ |

## سپورٹڈ ای میل زبانیں

Nudgen AI سے تیار کردہ ای میلز کے لیے زبانوں کی ایک وسیع رینج کو سپورٹ کرتا ہے۔ آپ **Settings** ← **Brand** میں ڈیفالٹ سیٹ کر سکتے ہیں، یا ہر مہم کے لیے اسے انفرادی طور پر تبدیل کر سکتے ہیں۔

فی الحال سپورٹڈ زبانیں:

| زبان | مقامی لیبل | کوڈ |
| :--- | :--- | :--- |

| **Spanish** | Español | `es` |
| **French** | Français | `fr` |
| **Chinese** | 中文 | `zh` |

| **Vietnamese** | Tiếng Việt | `vi` |
| **Korean** | 한국어 | `ko` |
| **German** | Deutsch | `de` |
| **Portuguese** | Português | `pt` |
| **Italian** | Italiano | `it` |
| **Hindi** | हिन्दी | `hi` |

<Tip>
اگر آپ کی ہدف شدہ زبان فہرست میں نہیں ہے تو ہماری سپورٹ ٹیم سے رابطہ کریں۔ ہم مسلسل مزید زبانوں کے لیے سپورٹ شامل کر رہے ہیں۔
</Tip>
