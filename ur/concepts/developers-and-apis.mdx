---
title: "ڈویلپرز اور APIs"
description: "انٹیگریشن اور API کی دستیابی"
---

Nudgen فی الحال کوئی عوامی API، ویب ہکس (webhooks)، یا ایمبیڈ ایبل SDK پیش نہیں کرتا ہے۔ تمام استعمال [nudgen.net](https://nudgen.net) پر ویب ایپ کے ذریعے ہوتا ہے۔ اگر آپ کو پروگرام کے ذریعے رسائی یا انٹیگریشنز کی ضرورت ہے تو ہم سے رابطہ کریں—ہم مستقبل کے ورژن میں APIs شامل کر سکتے ہیں۔
