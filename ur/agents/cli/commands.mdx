## جائزہ

Nudgen CLI میں دستیاب بنیادی کمانڈز دیکھیں۔ زیادہ تر کمانڈز ریسورس کی قسم (مہمات، رابطے، ٹیمیں وغیرہ) کے لحاظ سے گروپ کی گئی ہیں۔

### توثیق اور شناخت

| کمانڈ | تفصیل |
| :--- | :--- |
| `nudgen login` | اپنے پرسنل ایکسیس ٹوکن (PAT) کے ساتھ توثیق کریں۔ |
| `nudgen whoami` | موجودہ ٹوکن کے مالک اور صارف کی معلومات دکھائیں۔ |
| `nudgen logout` | مقامی اسناد (credentials) کو صاف کریں۔ |

### ٹیم مینجمنٹ

| کمانڈ | تفصیل |
| :--- | :--- |
| `nudgen teams list` | ان تمام ٹیموں کی فہرست بنائیں جن کا آپ حصہ ہیں۔ |
| `nudgen teams current` | فی الحال فعال ٹیم کا سیاق و سباق دکھائیں۔ |
| `nudgen teams switch <id>` | کسی مخصوص ٹیم پر سوئچ کریں۔ |

### مہمات (Campaigns)

فعال ٹیم کے اندر مارکیٹنگ کی کوششوں کا نظم کریں۔

| کمانڈ | تفصیل |
| :--- | :--- |
| `nudgen campaigns list` | موجودہ ٹیم میں فعال مہمات کی فہرست بنائیں۔ |
| `nudgen campaigns create` | انٹرایکٹو تخلیق وزرڈ۔ |
| `nudgen campaigns update <id>` | موجودہ مہم کی ترتیبات میں ترمیم کریں۔ |
| `nudgen campaigns delete <id>` | مہم کا ریکارڈ حذف کریں۔ |

### رابطے (Lead Management)

| کمانڈ | تفصیل |
| :--- | :--- |
| `nudgen contacts list` | موجودہ ٹیم میں تمام لیڈز کی فہرست بنائیں۔ |
| `nudgen contacts create` | دستی طور پر ایک نیا لیڈ ریکارڈ شامل کریں۔ |
| `nudgen contacts import` | بڑی تعداد میں ڈیٹا درآمد کرنے کے لیے انٹرایکٹو CSV میپر۔ |
| `nudgen contacts update <id>` | لیڈ کی تفصیلات میں ترمیم کریں۔ |
| `nudgen contacts delete <id>` | رابطے کا ریکارڈ حذف کریں۔ |

### برانڈ کی شناخت (Brand Identities)

| کمانڈ | تفصیل |
| :--- | :--- |
| `nudgen brand list` | ٹیم کے لیے تمام برانڈ آوازوں (brand voices) کی فہرست بنائیں۔ |
| `nudgen brand create` | نئی برانڈ آواز شروع کرنے کے لیے وزرڈ۔ |
| `nudgen brand update <id>` | برانڈ کنفیگریشنز میں ترمیم کریں۔ |
| `nudgen brand delete <id>` | برانڈ کی شناخت حذف کریں۔ |

### ریفرلز اور آمدنی

| کمانڈ | تفصیل |
| :--- | :--- |
| `nudgen referral stats` | ریفرل کی کارکردگی کا خلاصہ دیکھیں۔ |
| `nudgen referral link` | اپنا منفرد ریفرل لنک حاصل کریں۔ |
| `nudgen referral activity` | حالیہ کلک ہسٹری اور سائن اپس دیکھیں۔ |
| `nudgen referral referrals` | آپ کے ریفر کردہ ٹیموں اور ان کی حیثیت کی فہرست دیکھیں۔ |
| `nudgen referral payouts` | ادائیگی کی درخواستوں کی تاریخ اور حیثیت دیکھیں۔ |

اگلا، دیکھیں کہ ان کمانڈز کو [AI ایجینٹس](/ur/agents/cli/agentic-usage) کے لیے کیسے استعمال کیا جائے۔
