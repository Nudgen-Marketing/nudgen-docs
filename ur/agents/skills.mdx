## دستیاب اسکلز

### CLI اسکل
Nudgen CLI کو انسٹال اور توثیق کریں، ٹیموں، رابطوں، مہمات، برانڈ کی شناختوں، اور ریفرل ٹریکنگ کا انتظام کریں۔

```bash
npx skills add https://github.com/Nudgen-Marketing/skills --global --skill cli
```

**تفصیلات**: انسٹال کریں، PAT کے ذریعے توثیق کریں، اور ٹرمینل سے رابطوں، مہمات، برانڈ کی شناختوں اور ریفرل ڈیٹا کا انتظام کریں۔ ملٹی ٹیم کنٹیکسٹ سوئچنگ اور اسکرپٹنگ کے لیے `--json` آؤٹ پٹ کی حمایت کرتا ہے۔

---

### API اسکل
رابطوں کی CRUD، مہمات، ٹیمیں، برانڈ کنفیگریشنز، اعداد و شمار، ایفی لیٹ ڈیٹا، SDK پیٹرنز، ریٹ لمٹس، اور آئیڈیمپوٹینسی کیز (idempotency keys)۔

```bash
npx skills add https://github.com/Nudgen-Marketing/skills --global --skill api
```

**تفصیلات**: رابطوں، مہمات، ٹیمپلیٹس، برانڈ سیٹنگز، ٹیم مینجمنٹ، SDK بمقابلہ REST API پیٹرنز، ریٹ لمٹس، آئیڈیمپوٹینسی کیز، اور ایرر ہینڈلنگ کا احاطہ کرتا ہے۔ توثیق کے لیے `Authorization: Bearer <PAT>` استعمال کرتا ہے۔

---

### ای میل بہترین طریقوں کی اسکل
عام مقصد کی ای میل اسکل۔ ڈیلیوریبلٹی، ڈھانچہ، اور لائف سائیکل پیٹرنز کا آڈٹ کرتی ہے۔ Nudgen کے ساتھ یا اس کے بغیر کام کرتی ہے۔

```bash
npx skills add https://github.com/Nudgen-Marketing/skills --global --skill email-sending-best-practices
```

**تفصیلات**: بھیجنے والے کی توثیق (SPF، DKIM، DMARC) کی تصدیق کرتی ہے، ٹرانزیکشنل بمقابلہ مارکیٹنگ علیحدگی کو چیک کرتی ہے، ان سبسکرائب ہینڈلنگ کا جائزہ لیتی ہے، غائب لائف سائیکل مراحل کی نشاندہی کرتی ہے، اور اسپیم محرکات کے لیے مواد کو اسکور کرتی ہے۔

## تعاون کریں

ہم اپنی اسکلز لائبریری میں کمیونٹی کے تعاون کا خیرمقدم کرتے ہیں۔ اگر آپ نے Nudgen یا عام مارکیٹنگ آٹومیشن کے لیے کوئی مفید اسکل بنائی ہے، تو براہ کرم ہماری [GitHub ریپوزٹری](https://github.com/Nudgen-Marketing/skills) پر PR کھولیں۔
