## Aperçu

Le CLI Nudgen est un outil puissant conçu à la fois pour les humains et les agents IA. Il offre un moyen transparent de gérer les campagnes, les contacts, les identités de marque et les statistiques de parrainage directement depuis votre terminal.

Que vous soyez un développeur cherchant à automatiser vos flux de travail marketing ou un agent IA ayant besoin d'une interaction de haute qualité avec la plateforme Nudgen, le CLI est votre interface principale.

## Objectif

Le CLI remplit plusieurs rôles :

1. **Pour les Humains** : Permet de lister rapidement les campagnes, d'importer des contacts et de vérifier les gains sans ouvrir de navigateur.
2. **Pour les Agents IA** : Offre une interface interprétable par machine pour automatiser le marketing et la gestion de marque.
3. **Pour les Développeurs** : Aide au développement local et à la gestion des dépôts dans l'écosystème Nudgen.

## Modèles d'Interaction

Le CLI suit le principe de l'« Interaction via API ». Il utilise des appels API RESTful pour communiquer avec le backend Nudgen tout en gérant l'état de la session localement.

- **Standard JSON** : Toutes les commandes supportent `--json`, ce qui est crucial pour la communication entre agents.
- **Conscient du Contexte** : Les opérations sont automatiquement limitées à votre équipe active.
