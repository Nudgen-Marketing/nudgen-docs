## Installation Rapide

Le moyen le plus simple d'installer le CLI Nudgen est d'utiliser notre script d'installation en une ligne.

### macOS & Linux

Exécutez la commande suivante dans votre terminal :

```bash
curl -fsSL https://raw.githubusercontent.com/Nudgen-Marketing/nudgen-cli/main/scripts/install.sh | bash
```

### À partir des sources

Si vous avez Go 1.22+ installé, vous pouvez compiler à partir des sources :

```bash
git clone https://github.com/Nudgen-Marketing/nudgen-cli.git
cd nudgen-cli
make build
```

Le binaire sera disponible dans `./bin/nudgen`.

## Configuration Initiale

Une fois installé, vérifiez l'installation en consultant la version :

```bash
nudgen version
```

### Authentification

Pour commencer à utiliser le CLI, vous devez vous authentifier avec un jeton d'accès personnel (PAT).

1. Exécutez la commande de connexion :
   ```bash
   nudgen login
   ```
2. Suivez les instructions pour saisir votre jeton.
3. Vérifiez votre identité :
   ```bash
   nudgen whoami
   ```

### Mise à jour

Pour mettre à jour vers la dernière version, relancez simplement le script d'installation :

```bash
nudgen update
```

## Sécurité

Le CLI Nudgen utilise le trousseau système pour stocker votre PAT en toute sécurité. Nous n'affichons jamais de jetons bruts dans la console.

Ensuite, explorez les [commandes de base](/fr/cli/commands).
