## Aperçu

Découvrez les commandes de base disponibles dans le CLI Nudgen. La plupart sont regroupées par type de ressource (campagnes, contacts, équipes, etc.).

### Authentification & Identité

| Commande | Description |
| :--- | :--- |
| `nudgen login` | S'authentifier avec votre jeton d'accès personnel (PAT). |
| `nudgen whoami` | Afficher le propriétaire du jeton et les infos utilisateur. |
| `nudgen logout` | Effacer les identifiants locaux. |

### Gestion d'Équipe

| Commande | Description |
| :--- | :--- |
| `nudgen teams list` | Lister toutes les équipes auxquelles vous appartenez. |
| `nudgen teams current` | Afficher le contexte de l'équipe active. |
| `nudgen teams switch <id>` | Basculer vers une équipe spécifique. |

### Campagnes

Gérez vos efforts de marketing au sein de l'équipe active.

| Commande | Description |
| :--- | :--- |
| `nudgen campaigns list` | Lister les campagnes actives dans l'équipe actuelle. |
| `nudgen campaigns create` | Assistant de création interactif. |
| `nudgen campaigns update <id>` | Modifier les paramètres d'une campagne existante. |
| `nudgen campaigns delete <id>` | Supprimer un enregistrement de campagne. |

### Contacts (Gestion des prospects)

| Commande | Description |
| :--- | :--- |
| `nudgen contacts list` | Lister tous les prospects de l'équipe actuelle. |
| `nudgen contacts create` | Ajouter manuellement un nouveau prospect. |
| `nudgen contacts import` | Mappeur CSV interactif pour l'importation en masse. |
| `nudgen contacts update <id>` | Modifier les détails d'un prospect. |
| `nudgen contacts delete <id>` | Supprimer un prospect. |

### Identités de Marque

| Commande | Description |
| :--- | :--- |
| `nudgen brand list` | Lister toutes les voix de marque de l'équipe. |
| `nudgen brand create` | Assistant pour initialiser une nouvelle voix. |
| `nudgen brand update <id>` | Modifier les configurations de marque. |
| `nudgen brand delete <id>` | Supprimer une identité de marque. |

### Parrainage & Gains

| Commande | Description |
| :--- | :--- |
| `nudgen referral stats` | Voir le résumé des performances de parrainage. |
| `nudgen referral link` | Obtenir votre lien de parrainage unique. |
| `nudgen referral activity` | Voir l'historique des clics et inscriptions. |
| `nudgen referral referrals` | Lister les équipes parrainées et leur statut. |
| `nudgen referral payouts` | Voir l'historique des demandes de paiement. |

Ensuite, voyez comment utiliser ces commandes pour les [Agents IA](/fr/cli/agentic-usage).
