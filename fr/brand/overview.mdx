## Ce que cette page vous aide à faire

Définir la voix et le style des emails générés par l’IA afin qu’ils ressemblent à votre marque, et non à un modèle générique. Vous pouvez utiliser des presets, ajouter des instructions personnalisées et fournir des emails d’exemple.

## Avant de commencer

- Un compte Nudgen
- Une idée claire du ton que vous voulez adopter, par exemple décontracté, professionnel ou urgent

## Tons prédéfinis

Choisissez l’un des quatre presets pour orienter le style de l’IA :

| Preset | Idéal pour | Ambiance typique |
| ------ | ---------- | ---------------- |
| **Friendly** | SaaS, communautés, produits grand public | Chaleureux, accessible, conversationnel |
| **Professional** | B2B, finance, contextes formels | Clair, soigné, respectueux |
| **Playful** | Applications grand public, relances à la Duolingo | Léger, amusant, avec une urgence amicale |
| **Urgent** | Offres limitées, campagnes de reconquête | Direct, orienté action, avec sentiment d’urgence |

Vous pouvez modifier le preset à tout moment. Les nouvelles campagnes utilisent le preset actif, sauf si vous choisissez une autre configuration de marque.

## Prompt personnalisé

Ajoutez des instructions supplémentaires pour que l’IA colle davantage à votre ton. Exemples :

- "Utilisez des phrases courtes. Évitez le jargon."
- "Mentionnez le prénom de l’utilisateur une fois dans le premier paragraphe."
- "Terminez avec un seul CTA clair. Pas de liens multiples."
- "Le ton doit ressembler à celui d’un collègue utile, pas à un commercial."

Ces consignes sont combinées avec le preset et l’objectif de campagne.

## Emails d’exemple

Fournissez 1 à 3 emails d’exemple qui reflètent bien le style attendu. L’IA les utilise comme référence. Pensez à inclure :

- Des objets que vous aimez
- La manière d’ouvrir et de conclure
- Le niveau de formalité
- Le style des CTA

<Note>
Les exemples servent uniquement à calibrer le ton. L’IA ne copie pas vos textes ; elle en retient le style pour produire du contenu neuf.
</Note>

## Plusieurs configurations de marque

Vous pouvez enregistrer plusieurs configurations, par exemple "Lancement produit", "Réengagement" ou "Support". Lors de la création d’une campagne, choisissez celle qui convient le mieux.

## Ce qui se passe ensuite

- [Créez une campagne](/fr/campaigns/create) et sélectionnez votre configuration de marque.
- Si le résultat ne vous convient pas, ajustez le prompt puis régénérez le contenu.
