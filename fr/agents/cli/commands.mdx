## Aperçu

Explorez les commandes de base disponibles dans la CLI Nudgen. La plupart des commandes sont regroupées par type de ressource (campagnes, contacts, équipes, etc.).

### Authentification & Identité

| Commande | Description |
| :--- | :--- |
| `nudgen login` | S'authentifier avec votre jeton d'accès personnel (PAT). |
| `nudgen whoami` | Afficher le propriétaire du jeton actuel et les infos utilisateur. |
| `nudgen logout` | Effacer les identifiants locaux. |

### Gestion d'équipe

| Commande | Description |
| :--- | :--- |
| `nudgen teams list` | Lister toutes les équipes auxquelles vous appartenez. |
| `nudgen teams current` | Afficher le contexte de l'équipe actuellement active. |
| `nudgen teams switch <id>` | Basculer vers une équipe spécifique. |

### Campagnes

Gérez les efforts de marketing au sein de l'équipe active.

| Commande | Description |
| :--- | :--- |
| `nudgen campaigns list` | Lister les campagnes actives dans l'équipe actuelle. |
| `nudgen campaigns create` | Assistant de création interactif. |
| `nudgen campaigns update <id>` | Modifier les paramètres d'une campagne existante. |
| `nudgen campaigns delete <id>` | Supprimer un enregistrement de campagne. |

### Contacts (Gestion des prospects)

| Commande | Description |
| :--- | :--- |
| `nudgen contacts list` | Lister tous les prospects de l'équipe actuelle. |
| `nudgen contacts create` | Ajouter manuellement un nouvel enregistrement de prospect. |
| `nudgen contacts import` | Mappeur CSV interactif pour l'ingestion massive. |
| `nudgen contacts update <id>` | Modifier les détails d'un prospect. |
| `nudgen contacts delete <id>` | Supprimer un enregistrement de contact. |

### Identités de marque

| Commande | Description |
| :--- | :--- |
| `nudgen brand list` | Lister toutes les voix de marque pour l'équipe. |
| `nudgen brand create` | Assistant pour initialiser une nouvelle voix de marque. |
| `nudgen brand update <id>` | Modifier les configurations de marque. |
| `nudgen brand delete <id>` | Supprimer une identité de marque. |

### Parrainages & Gains

| Commande | Description |
| :--- | :--- |
| `nudgen referral stats` | Voir le résumé des performances de parrainage de haut niveau. |
| `nudgen referral link` | Obtenir votre lien de parrainage unique. |
| `nudgen referral activity` | Voir l'historique récent des clics et des inscriptions. |
| `nudgen referral referrals` | Lister les équipes que vous avez parrainées et leur statut. |
| `nudgen referral payouts` | Voir l'historique des demandes de paiement et leur statut. |

Ensuite, découvrez comment utiliser ces commandes pour les [agents d'IA](/fr/agents/cli/agentic-usage).
