## Aperçu

La CLI Nudgen est un outil puissant conçu à la fois pour les humains et les agents d'IA. Elle offre un moyen fluide de gérer les campagnes, les contacts, les identités de marque et les statistiques de parrainage directement depuis votre terminal.

Que vous soyez un développeur cherchant à automatiser vos flux de travail marketing ou un agent d'IA ayant besoin d'une interaction de haute qualité avec la plateforme Nudgen, la CLI est votre interface principale.

## Objectif

La CLI remplit plusieurs rôles :

1. **Pour les humains** : Fournit un moyen rapide de lister les campagnes, d'importer des contacts et de vérifier les revenus de parrainage sans ouvrir le navigateur web.
2. **Pour les agents d'IA** : Offre une interface interprétable par machine pour automatiser les campagnes marketing complexes et la gestion de marque.
3. **Pour les développeurs** : Aides pour le développement local et la gestion de dépôts au sein de l'écosystème Nudgen.

## Modèles d'interaction

La CLI suit le principe d'« Interaction sur API ». Elle utilise des appels API RESTful standard pour communiquer avec le backend Nudgen tout en gérant l'état de la session localement de manière sécurisée.

- **Standard JSON** : Toutes les commandes prennent en charge `--json`. C'est essentiel pour la communication entre agents.
- **Sensible au contexte** : Les opérations comme le listage des campagnes sont automatiquement limitées à votre équipe active.
