Après avoir créé votre signature dans [l'éditeur](/fr/extension/signature-usage), vous devez l'installer dans votre client de messagerie pour commencer à l'utiliser.

## Clients de messagerie courants

> Suivez [ce lien](https://pdf.wondershare.com/signature/insert-html-signature-in-gmail.html) pour ajouter une signature à Gmail. Consultez [ceci](https://www.christopherbolt.com/support/knowledgebase/24/Installing-HTML-email-signatures-in-Microsoft-Outlook.html) pour installer une signature HTML dans Outlook.

## Étapes générales d'installation

Si vous utilisez un autre client de messagerie, suivez ces étapes générales :

1.  Dans l'aperçu du générateur de signature, cliquez sur **Copier dans le presse-papier**.
2.  Ouvrez les **Paramètres** ou **Préférences** de votre messagerie.
3.  Localisez la section **Signature** (souvent sous "Mail" ou "Composition").
4.  S'il existe déjà une signature, supprimez-la ou créez-en une nouvelle.
5.  Collez (`Ctrl + V` ou `Cmd + V`) votre signature copiée dans l'éditeur.
6.  **Enregistrez** vos paramètres.

<Tip>
  Si votre signature semble légèrement différente selon les clients, assurez-vous que votre messagerie est configurée pour envoyer des e-mails au format **HTML/Texte enrichi** plutôt qu'en texte brut.
</Tip>
