## Ce que cette page vous aide à faire

Trouver et gérer les paramètres liés à votre compte. Cette page vous oriente vers les bons écrans pour le profil, l’équipe, les signatures et la configuration de marque.

## Avant de commencer

- Un compte Nudgen
- Un rôle Owner ou Admin

## Vue d’ensemble des paramètres

Dans **Settings**, vous pouvez accéder à :

| Section | Ce que vous pouvez faire | Doc |
| ------- | ------------------------ | --- |
| **Team** | Inviter des membres, attribuer des rôles et gérer les permissions | [Team settings](/fr/settings/team) |
| **Brand** | Définir l’identité de marque : nom, description, reply-to, logo | [Brand overview](/fr/brand/overview) |
| **Signatures** | Créer et modifier les signatures utilisées dans les campagnes | [Signatures](/fr/settings/signatures) |

## Workspace et compte

Votre compte peut appartenir à plusieurs workspaces. Le workspace actif est affiché dans l’en-tête de la barre latérale. Toutes les données, comme les contacts, campagnes, configurations de marque et la facturation, sont liées à ce workspace actif.
