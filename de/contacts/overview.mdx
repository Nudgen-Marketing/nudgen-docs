## Was diese Seite Ihnen bietet

Erfahren Sie, wie Kontakte gespeichert werden, warum sie maskiert erscheinen und wie Sie Tags zur Segmentierung verwenden. Dies bereitet Sie darauf vor, Kontakte sicher hochzuladen und Kampagnen zu erstellen.

## Bevor Sie beginnen

- Ein Nudgen-Konto
- Eine CSV-Datei mit mindestens einer E-Mail-Adresse (siehe [Upload-Anleitung](/de/contacts/upload) für das Format)

## Wie Kontaktdaten gespeichert werden

Nudgen verschlüsselt jede Art von personenbezogenen Daten (PII), bevor sie gespeichert werden:

- **E-Mails** — AES-256 verschlüsselt. Wir können sie nicht lesen.
- **Namen** — Verschlüsselt, falls angegeben. Optional in Ihrer CSV.
- **Benutzerdefinierte Metadaten** — Alle zusätzlichen Spalten, die Sie hochladen, werden ebenfalls verschlüsselt.

Wir speichern zudem einen Einweg-Hash jeder E-Mail zur Duplikaterkennung. Doppelte E-Mails in derselben Liste werden automatisch zusammengeführt.

## Maskierte Ansicht

Standardmäßig zeigt die Kontaktliste maskierte Werte wie `j***@g***.com` anstelle der echten Adressen an. PII sind im Ruhezustand verschlüsselt; wir sehen oder speichern niemals Klartext-E-Mails oder -Namen. Die Maskierung schützt Ihre Daten vor Schulterblicken (Shoulder-Surfing) und begrenzt die Bloßstellung, falls jemand anderes Zugriff auf Ihren Bildschirm hat.

**Einen Kontakt enthüllen:** Sie (und andere Workspace-Mitglieder mit Zugriff) können auf **Reveal** (Enthüllen) klicken, um die echte E-Mail und den Namen in der UI anzuzeigen. Die Entschlüsselung findet nur in Ihrem Browser statt, wenn Sie darauf klicken; sobald Sie die Seite verlassen oder aktualisieren, werden die Werte wieder maskiert. Wir zeigen niemals entschlüsselte PII in Analysen, Protokollen oder Exporten an.

## Tags und Segmentierung

Weisen Sie Tags zu, wenn Sie Kontakte hochladen oder sie später bearbeiten. Nutzen Sie Tags, um spezifische Segmente in Kampagnen anzusprechen:

- `inactive` — Benutzer, die sich seit 30+ Tagen nicht angemeldet haben
- `vip` — Besonders wertvolle oder Premium-Kunden
- `trial` — Personen in einem kostenlosen Testzeitraum

Wenn Sie eine Kampagne erstellen, wählen Sie eine Zielgruppe: alle Kontakte oder nur solche mit bestimmten Tags. Tags machen es einfach, unterschiedliche Sequenzen für verschiedene Segmente auszuführen.

## Kontakt-Status

Kontakte können einen dieser Status haben:

- **Aktiv** — Kann Kampagnen erhalten. Standard für neu importierte Kontakte.
- **Abgemeldet** — Über den Abmelde-Link oder manuell abgemeldet. Sie erhalten keine weiteren Kampagnen-E-Mails, es sei wenn Sie den Status ändern.
- **Unterdrückt / Bounced** — Zustellung fehlgeschlagen (ungültige Adresse, Postfach voll oder Provider-Sperrliste). Diese Kontakte werden automatisch von zukünftigen Sendungen ausgeschlossen, um die Zustellbarkeit zu schützen. Unser System synchronisiert sich mit der Sperrliste des Zustellanbieters.
