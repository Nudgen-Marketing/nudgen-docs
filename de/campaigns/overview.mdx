## Was diese Seite Ihnen bietet

Verstehen Sie die zwei Kampagnentypen und was jeder Status bedeutet. Dies hilft Ihnen, den richtigen Kampagnentyp zu wählen und Ihre Kampagnenliste zu interpretieren.

## Bevor Sie beginnen

- Kontakte müssen in Nudgen hochgeladen sein
- Eine Markenkonfiguration (vordefinierte Tonalität oder benutzerdefinierter Prompt)

## Kampagnentypen

### Einmalige Kampagnen (One-shot)

Eine einzelne E-Mail, die einmal an Ihre ausgewählte Zielgruppe gesendet wird. Nutzen Sie diese, um:

- Eine Funktion oder ein Event anzukündigen
- Eine einmalige Reaktivierungs-Kampagne zu senden
- Eine Nachricht zu testen, bevor Sie eine längere Sequenz erstellen

Sie wählen die Zielgruppe, die KI generiert den Inhalt, Sie prüfen die Vorschau und senden (oder planen) die Nachricht.

### Drip-Sequenzen

Eine mehrstufige Kampagne mit zeitlichen Verzögerungen zwischen den E-Mails (z. B. Tag 1, Tag 3, Tag 7). Nutzen Sie diese, um:

- Inaktive Benutzer über mehrere Tage hinweg anzustupsen (Nudge)
- Follow-ups an Personen zu senden, die nicht geöffnet oder geklickt haben
- Einen Retention-Flow im "Duolingo-Stil" aufzubauen

Jeder Schritt hat sein eigenes Ziel, seinen eigenen Betreff und einen KI-generierten Text. Kontakte bewegen sich erst nach der Verzögerung zum nächsten Schritt. Sie können **Auto-Stop-Regeln** festlegen (z. B. Stopp bei Klick oder bei Öffnen der E-Mail), damit Benutzer, die bereits interagieren, keine weiteren Schritte erhalten.

**Zielgruppen-Typen:** Beim Erstellen einer Kampagne wählen Sie, wer sie erhalten soll: **Alle Kontakte**, **Nach Tags** (nur Kontakte mit ausgewählten Tags) oder **Manuell** (einzelne Empfänger auswählen). Der Assistent zeigt eine Reichweitenschätzung an, bevor Sie starten.

## Kampagnen-Status

| Status     | Bedeutung                                                              |
| ---------- | ---------------------------------------------------------------------- |
| **Entwurf (Draft)** | Nicht gesendet. Sie können Ziele, Zielgruppe, Inhalt und Zeitplan bearbeiten. |
| **Geplant (Scheduled)** | Sendezeitpunkt ist festgelegt. E-Mails werden zur geplanten Zeit versendet. |
| **Aktiv / Senden (Active)** | E-Mails werden aktuell versendet. Der Fortschritt erscheint im Dashboard. |
| **Gesendet (Sent)** | Alle E-Mails wurden versendet. Der Inhalt ist gesperrt; Sie können Analysen einsehen. |
| **Gestoppt (Stopped)** | Die Kampagne wurde manuell gestoppt. Einige Kontakte haben evtl. bereits E-Mails erhalten; zukünftige Drip-Schritte sind abgebrochen. |

<Note>
Sobald eine Kampagne gesendet wurde, können Sie deren Inhalt nicht mehr bearbeiten oder sie erneut senden. Erstellen Sie eine neue Kampagne, um eine andere Nachricht zu senden.
</Note>
