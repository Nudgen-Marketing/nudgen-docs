## Was diese Seite Ihnen bietet

Bearbeiten Sie Kampagnen-Entwürfe, senden oder planen Sie den Versand und stoppen Sie aktive Kampagnen. Erfahren Sie zudem, was Sie nach dem Versenden einer Kampagne noch ändern können und was nicht.

## Bevor Sie beginnen

- Mindestens eine Kampagne (Entwurf oder gesendet) vorhanden
- Rolle als Inhaber oder Teammitglied mit Bearbeitungsrechten

## Eine Kampagne bearbeiten

**Sie können nur Entwürfe bearbeiten.** Sobald eine Kampagne gesendet wurde, sind Inhalt und Zielgruppe gesperrt.

Bei Entwürfen können Sie folgendes ändern:

- Zielgruppe (alle Kontakte oder Tags)
- Markenkonfiguration (Markenton)
- Kampagnenziel und KI-generierter Inhalt
- Zeitplan (Datum und Uhrzeit)
- Auto-Stop-Regeln (für Drip-Sequenzen)

1. Gehen Sie zu **Kampagnen**.
2. Öffnen Sie die Kampagne (Klick auf den Namen oder **Bearbeiten**).
3. Nehmen Sie die Änderungen im Assistenten vor.
4. Speichern. Falls Sie die Kampagne geplant hatten, gilt der neue Zeitplan.

## Sofort senden

1. Öffnen Sie eine Entwurfs-Kampagne.
2. Klicken Sie auf **Jetzt senden (Send now)**.
3. Bestätigen Sie. E-Mails werden in die Warteschlange gestellt und innerhalb weniger Minuten versendet.

<Note>
Das Senden ist unumkehrbar. Überprüfen Sie Zielgruppe und Inhalt sorgfältig, bevor Sie bestätigen.
</Note>

## Für später planen

1. Öffnen Sie eine Entwurfs-Kampagne.
2. Wählen Sie **Planen (Schedule)** und legen Sie Datum sowie Uhrzeit fest.
3. Speichern. Der Kampagnenstatus wechselt zu **Geplant (Scheduled)**.
4. Zum geplanten Zeitpunkt werden die E-Mails automatisch versendet.

Sie können eine geplante Kampagne bearbeiten oder abbrechen, bevor sie ausgeführt wird.

## Eine Kampagne stoppen

Verwenden Sie **Stop campaign**, wenn Sie den Versand sofort anhalten möchten. Dies gilt für:

- Kampagnen mit Status **Senden (Sending)**
- **Geplante** Kampagnen (bricht den Versand ab)
- **Drip-Sequenzen** — stoppt zukünftige Schritte für Kontakte, die diese noch nicht erhalten haben

Bereits versendete E-Mails können nicht zurückgerufen werden. Empfänger, die vor dem Stopp eine E-Mail erhalten haben, behalten diese in ihrem Posteingang.

## Entwurf vs. Gesendet: Was sich ändern lässt

| Aktion                    | Entwurf | Geplant | Gesendet |
| ------------------------- | :-----: | :-----: | :------: |
| Inhalt bearbeiten         | Ja      | Ja      | Nein     |
| Zielgruppe ändern         | Ja      | Ja      | Nein     |
| Zeitplan ändern           | Ja      | Ja      | N/A      |
| Jetzt senden              | Ja      | Ja      | N/A      |
| Stoppen                   | N/A     | Ja      | Ja*      |
| Analyse ansehen           | Nein    | Nein    | Ja       |

\*Das Stoppen einer gesendeten Kampagne verhindert nur zukünftige Schritte (z. B. in einem Drip). Vergangene E-Mails sind nicht betroffen.

## Nächste Schritte

- Prüfen Sie nach dem Senden die [Kampagnen-Analyse](/de/campaigns/analytics) auf Zustellraten, Öffnungen und Klicks.
- Erstellen Sie eine neue Kampagne, wenn Sie eine andere Nachricht senden möchten.
