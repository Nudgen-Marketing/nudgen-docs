## Was diese Seite Ihnen bietet

Erstellen Sie wiederverwendbare E-Mail-Signaturen und hängen Sie diese an Kampagnen an. Signaturen erscheinen am Ende Ihrer KI-generierten E-Mails, damit Empfänger Ihren Namen, Ihren Titel und Ihre Kontaktdaten sehen können.

## Bevor Sie beginnen

- Ein Nudgen-Konto
- Zugriff als Inhaber, Administrator oder Mitglied

## Eine Signatur erstellen

1. Gehen Sie zu **Einstellungen** → **Signaturen**.
2. Klicken Sie auf **Neue Signatur (New signature)** oder **Signatur hinzufügen (Add signature)**.
3. Geben Sie einen **Namen** ein (z. B. "Standard", "Vertrieb", "Support"), damit Sie sie bei der Auswahl identifizieren können.
4. Fügen Sie Ihren Signatur-Inhalt hinzu. Sie können folgendes nutzen:
   - Nur-Text
   - Einfache Formatierung (fett, Links, Zeilenumbrüche)
   - HTML, falls unterstützt
5. Speichern.

## Verwendung in Kampagnen

Wählen Sie beim Erstellen oder Bearbeiten einer Kampagne aus, welche Signatur angehängt werden soll. Die Signatur wird beim Versenden der Kampagne an den E-Mail-Text angehängt. Sie können mehrere Signaturen speichern und die passende per Kampagne wählen (z. B. Vertrieb vs. Support).

## Bearbeiten oder Löschen

- **Bearbeiten:** Öffnen Sie die Signatur, ändern Sie den Inhalt oder Namen und speichern Sie.
- **Löschen:** Entfernen Sie die Signatur. Kampagnen, die sie bereits genutzt haben, behalten den gesendeten Inhalt; zukünftige Kampagnen müssen eine andere Signatur wählen.

<Warning>
Wenn Sie eine Signatur löschen, die noch in einem Kampagnen-Entwurf ausgewählt ist, müssen Sie vor dem Versenden eine neue Signatur auswählen.
</Warning>

## Inhaltliche Erwartungen und Sicherheit

- Halten Sie Signaturen kurz. Lange Signaturen können Spam-Filter auslösen oder unprofessionell wirken.
- Vermeiden Sie große Bilder oder komplexe Layouts. Einfacher Text mit ein oder zwei Links funktioniert am besten.
- Geben Sie Ihren Namen, Titel und eine Kontaktmöglichkeit (E-Mail oder Link) an. Optional: Unternehmensname, Logo.
- Signaturen werden vor dem Anhängen an E-Mails bereinigt (sanitized), um Injection-Angriffe zu verhindern. Nur sicheres HTML und Links sind erlaubt; riskante Inhalte werden entfernt.

## Nächste Schritte

- [Erstellen Sie eine Kampagne](/de/campaigns/create) und wählen Sie Ihre Signatur vor dem Versand aus.
- Erstellen Sie mehrere Signaturen für verschiedene Teams oder Anwendungsfälle.

## Fehlerbehebung

  <Accordion title="Signatur erscheint nicht in der E-Mail">
    Stellen Sie sicher, dass Sie die Signatur beim Erstellen oder Bearbeiten der Kampagne ausgewählt haben. Prüfen Sie die Vorschau vor dem Senden.
  </Accordion>
  <Accordion title="Formatierung sieht fehlerhaft aus">
    Nutzen Sie einfache Formatierungen. Komplexes HTML wird in verschiedenen E-Mail-Clients möglicherweise nicht konsistent dargestellt.
  </Accordion>
</AccordionGroup>
