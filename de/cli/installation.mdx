## Schnell-Installation

Der einfachste Weg, das Nudgen CLI zu installieren, ist unser Einzeiler-Installationsskript.

### macOS & Linux

Führen Sie den folgenden Befehl in Ihrem Terminal aus:

```bash
curl -fsSL https://raw.githubusercontent.com/Nudgen-Marketing/nudgen-cli/main/scripts/install.sh | bash
```

### Aus dem Quellcode

Wenn Sie Go 1.22+ installiert haben, können Sie das Paket direkt aus dem Quellcode bauen:

```bash
git clone https://github.com/Nudgen-Marketing/nudgen-cli.git
cd nudgen-cli
make build
```

Die Binärdatei ist dann unter `./bin/nudgen` verfügbar.

## Ersteinrichtung

Überprüfen Sie nach der Installation die Version:

```bash
nudgen version
```

### Authentifizierung

Um das CLI zu nutzen, müssen Sie sich mit einem persönlichen Zugriffstoken (Personal Access Token, PAT) authentifizieren.

1. Führen Sie den Login-Befehl aus:
   ```bash
   nudgen login
   ```
2. Folgen Sie den interaktiven Abfragen, um Ihr Token einzugeben.
3. Überprüfen Sie Ihre Identität:
   ```bash
   nudgen whoami
   ```

### Aktualisierung

Um auf die neueste Version zu aktualisieren, führen Sie einfach das Installationsskript erneut aus oder nutzen Sie den integrierten Update-Befehl (falls verfügbar):

```bash
nudgen update
```

## Sicherheit

Das Nudgen CLI nutzt den Schlüsselbund Ihres Systems (Keychain), um Ihr PAT sicher zu speichern. Wir protokollieren oder zeigen niemals unverschlüsselte Token in der Befehlsausgabe an.

Entdecken Sie als Nächstes die [Kernbefehle](/de/cli/commands).
