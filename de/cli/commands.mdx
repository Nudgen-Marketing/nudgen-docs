## Überblick

Lernen Sie die Kernbefehle kennen, die im Nudgen CLI verfügbar sind. Die meisten Befehle sind nach Ressourcentyp (Kampagnen, Kontakte, Teams usw.) gruppiert.

### Authentifizierung & Identität

| Befehl | Beschreibung |
| :--- | :--- |
| `nudgen login` | Authentifizierung mit Ihrem persönlichen Zugriffstoken (PAT). |
| `nudgen whoami` | Zeigt den aktuellen Token-Besitzer und die Benutzerinfos an. |
| `nudgen logout` | Löscht die lokalen Anmeldedaten. |

### Team-Verwaltung

| Befehl | Beschreibung |
| :--- | :--- |
| `nudgen teams list` | Listet alle Teams auf, denen Sie angehören. |
| `nudgen teams current` | Zeigt den aktuell aktiven Team-Kontext an. |
| `nudgen teams switch <id>` | Wechselt zu einem spezifischen Team. |

### Kampagnen

Verwalten Sie Ihre Marketingbemühungen innerhalb des aktiven Teams.

| Befehl | Beschreibung |
| :--- | :--- |
| `nudgen campaigns list` | Listet aktive Kampagnen im aktuellen Team auf. |
| `nudgen campaigns create` | Interaktiver Erstellungs-Assistent. |
| `nudgen campaigns update <id>` | Bestehende Kampagneneinstellungen ändern. |
| `nudgen campaigns delete <id>` | Ein Kampagnen-Datensatz löschen. |

### Kontakte (Lead-Management)

| Befehl | Beschreibung |
| :--- | :--- |
| `nudgen contacts list` | Listet alle Leads im aktuellen Team auf. |
| `nudgen contacts create` | Einen Lead-Eintrag manuell hinzufügen. |
| `nudgen contacts import` | Interaktiver CSV-Mapper für Massen-Importe. |
| `nudgen contacts update <id>` | Lead-Details bearbeiten. |
| `nudgen contacts delete <id>` | Einen Kontakt löschen. |

### Markenidentitäten (Brand)

| Befehl | Beschreibung |
| :--- | :--- |
| `nudgen brand list` | Listet alle Marken-Tonalitäten des Teams auf. |
| `nudgen brand create` | Assistent zum Initialisieren einer neuen Marken-Tonalität. |
| `nudgen brand update <id>` | Markenkonfigurationen bearbeiten. |
| `nudgen brand delete <id>` | Eine Markenidentität löschen. |

### Affiliate & Einnahmen

| Befehl | Beschreibung |
| :--- | :--- |
| `nudgen referral stats` | High-Level-Performance-Zusammenfassung der Affiliate-Aktivitäten. |
| `nudgen referral link` | Ihren individuellen Empfehlungslink abrufen. |
| `nudgen referral activity` | Letzte Klicks und Anmeldungen sehen. |
| `nudgen referral referrals` | Geworbene Teams und deren Status auflisten. |
| `nudgen referral payouts` | Historie und Status von Auszahlungsanfragen sehen. |
