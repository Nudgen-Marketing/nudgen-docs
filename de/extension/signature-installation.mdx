Nachdem Sie Ihre Signatur im [Editor](/de/extension/signature-usage) erstellt haben, müssen Sie sie in Ihrem E-Mail-Programm installieren, um sie verwenden zu können.

## Häufigste E-Mail-Programme

> Folgen Sie [diesem Link](https://pdf.wondershare.com/signature/insert-html-signature-in-gmail.html), um eine Signatur zu Gmail hinzuzufügen. Informationen zur Installation einer HTML-Signatur in Outlook finden Sie [hier](https://www.christopherbolt.com/support/knowledgebase/24/Installing-HTML-email-signatures-in-Microsoft-Outlook.html).

## Allgemeine Installationsschritte

Wenn Sie ein anderes E-Mail-Programm verwenden, folgen Sie diesen allgemeinen Schritten:

1.  Klicken Sie in der Vorschau des Signatur-Generators auf **In die Zwischenablage kopieren**.
2.  Öffnen Sie die **Einstellungen** oder **Optionen** Ihres E-Mail-Programms.
3.  Suchen Sie den Bereich **Signatur** (oft unter "E-Mail" oder "Verfassen").
4.  Löschen Sie eine vorhandene Signatur oder erstellen Sie eine neue.
5.  Fügen Sie Ihre kopierte Signatur ein (`Strg + V` oder `Cmd + V`).
6.  **Speichern** Sie Ihre Einstellungen.

<Tip>
  Falls Ihre Signatur in bestimmten Programmen leicht anders aussieht, stellen Sie sicher, dass Ihr Programm so konfiguriert ist, dass E-Mails im **HTML/Rich-Text-Format** und nicht als Nur-Text versendet werden.
</Tip>
