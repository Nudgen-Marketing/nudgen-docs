## Was diese Seite Ihnen bietet

Legen Sie die Stimme und den Stil der KI-generierten E-Mails fest, damit sie wie Ihre Marke klingen und nicht wie eine generische Vorlage. Sie können Vorlagen (Presets) verwenden, benutzerdefinierte Anweisungen hinzufügen und Beispiel-E-Mails zur Kalibrierung bereitstellen.

## Bevor Sie beginnen

- Ein Nudgen-Konto
- Eine klare Vorstellung davon, wie Sie klingen möchten (z. B. locker, professionell, dringlich)

## Tonalitäts-Vorlagen (Presets)

Wählen Sie eine von vier Vorlagen aus, um den Standardstil der KI zu bestimmen:

| Vorlage        | Bestens geeignet für                  | Beispiel-Vibe                        |
| -------------- | ------------------------------------- | ------------------------------------ |
| **Freundlich** | E-Commerce, Shops, Lifestyle-Produkte | Herzlich, nahbar, gesprächig         |
| **Professionell** | B2B, Finanzen, formelle Kontexte      | Klar, geschliffen, respektvoll       |
| **Verspielt**  | Consumer-Apps, "Duolingo-Stil" Nudges | Lustig, leichter Humor, freundlich-dringlich |
| **Dringlich**  | Zeitlich begrenzte Angebote, Winbacks | Direkt, handlungsorientiert, Knappheit |

Sie können die Vorlage jederzeit ändern. Neue Kampagnen verwenden das aktive Preset, es sei denn, Sie wählen eine andere Markenkonfiguration.

## Benutzerdefinierter Prompt

Fügen Sie zusätzliche Anweisungen hinzu, damit die KI Ihren Stil noch besser trifft. Beispiele:

- "Verwenden Sie kurze Sätze. Vermeiden Sie Fachjargon."
- "Sprechen Sie den Nutzer im ersten Absatz einmal mit seinem Namen an."
- "Beenden Sie die E-Mail mit einem einzigen klaren CTA. Keine multiplen Links."
- "Die Tonalität sollte wie ein hilfreicher Kollege wirken, nicht verkäuferisch."

Diese Anweisungen werden beim Generieren von Inhalten mit dem Preset und dem Kampagnenziel kombiniert.

## Beispiel-E-Mails

Geben Sie 1–3 Beispiel-E-Mails an, die den gewünschten Ton widerspiegeln. Die KI nutzt diese als Referenz beim Generieren neuer Inhalte. Geben Sie an:

- Betreffzeilen, die Ihnen gefallen
- Stil von Begrüßung und Verabschiedung
- Grad der Förmlichkeit
- Stil der Handlungsaufforderung (CTA)

<Note>
Beispiele dienen nur der Tonalitäts-Kalibrierung. Die KI kopiert Ihren Text nicht; sie lernt den Stil und wendet ihn auf neue Kampagnen an.
</Note>

## Mehrere Markenkonfigurationen

Sie können mehrere Markenkonfigurationen speichern (z. B. "Produkt-Launch", "Reaktivierung", "Support"). Wählen Sie beim Erstellen einer Kampagne aus, welche Konfiguration Sie nutzen möchten. So können Sie den Ton je nach Kampagnentyp wechseln, ohne eine globale Einstellung ändern zu müssen.

## Nächste Schritte

- [Erstellen Sie eine Kampagne](/de/campaigns/create) und wählen Sie Ihre Markenkonfiguration aus. Die KI verwendet Ihren Ton und Ihre Prompts.
- Generieren Sie den Inhalt neu, falls das Ergebnis nicht passt — Sie können den Prompt anpassen und es erneut versuchen.

## Fehlerbehebung

<AccordionGroup>
  <Accordion title="Generierte E-Mails passen nicht zu meinen Beispielen">
    Fügen Sie spezifischere Anweisungen im benutzerdefinierten Prompt hinzu. Stellen Sie sicher, dass Ihre Beispiele den gewünschten Ton (formell vs. locker, kurz vs. lang) klar zeigen.
  </Accordion>
  <Accordion title="Kann ich verschiedene Töne für verschiedene Kampagnen nutzen?">
    Ja. Erstellen Sie mehrere Markenkonfigurationen und wählen Sie eine pro Kampagne aus.
  </Accordion>
</AccordionGroup>
