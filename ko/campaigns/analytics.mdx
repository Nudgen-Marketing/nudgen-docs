## 이 페이지의 도움말
캠페인의 성과를 분석하는 방법을 설명합니다. 얼마나 많은 이메일이 발송되었고, 도달되었으며, 열어보거나 클릭했는지 확인할 수 있습니다. 또한 발송 성공 및 실패율, 수신자별 상세 활동 데이터도 제공합니다.

## 시작하기 전에
- 발송되었거나 발송 중인 캠페인이 최소 하나 이상 있어야 합니다.
- 해당 캠페인의 상세 페이지에 접근할 수 있어야 합니다.

## 지표 카드 (Metric cards)
캠페인 상세 페이지에서 다음 지표를 확인할 수 있습니다:

| 지표 | 의미 |
| :--- | :--- |
| **Emails Sent (발송 수)** | API 또는 네트워크 오류를 포함한 전체 발송 시도 횟수입니다. |
| **Emails Delivered (도달 수)** | 수신자의 메일 서버에서 성공적으로 수락된 이메일 수입니다. |
| **Total Opens (오픈 수)** | 추적 픽셀을 통해 이메일이 열린 전체 횟수입니다. |
| **Total Clicks (클릭 수)** | 본문 내 링크가 클릭된 전체 횟수입니다. |
| **CTR (클릭률)** | (클릭 수 / 발송 수) × 100입니다. |
| **Spam Complaints (스팸 신고)** | 수신자가 해당 이메일을 스팸으로 표시한 횟수입니다. |
| **Soft Bounce (일시적 실패)** | 일시적인 발송 문제(편지함 가득 참, 트래픽 제한, 일시적 서버 오류)입니다. |
| **Hard Bounce (영구적 실패)** | 영구적인 발송 실패(잘못된 주소, 존재하지 않는 도메인)입니다. |

## 발송 성공 및 실패 분석
- **성공률 (Success rate)** = (도달 수 / 발송 수) × 100
- **실패율 (Failure rate)** = (바운스 + 실패 건수) / 발송 수 × 100

실패율이 높으면 발신자 평판에 악영향을 줄 수 있습니다. 바운스된 주소는 목록에서 삭제하거나 수정하고, 유효하지 않은 연락처는 정리해 주세요.

## 시계열 활동 차트
일, 주, 월 단위로 캠페인 성과를 보여주는 차트입니다. 보기 방식을 변경하거나 개별 지표를 끄고 켜서 오픈, 클릭, 도달 추이에 집중하여 분석할 수 있습니다.

## 날짜 범위 선택기
특정 기간의 지표와 차트 데이터를 필터링할 수 있습니다. 기본 설정은 캠페인 전체 기간입니다. 특정 주간의 데이터를 비교하거나 최근 활동에 집중하고 싶을 때 유용합니다.

## 수신자 활동 내역 테이블
각 연락처별 상태를 상세히 확인할 수 있습니다:

- **Contact (연락처)** — 이메일(확인 전까지 마스킹 처리) 및 이름
- **Status (상태)** — 최근 활동: 발송, 도달, 오픈, 클릭, 바운스됨
- **Sent at (발송 시각)** — 이메일이 발송된 시각
- **Delivered at (도달 시각)** — 도달이 확인된 시각 (확인 가능한 경우)
- **Opened at (오픈 시각)** — 처음 메일을 열어본 시각
- **Clicked at (클릭 시각)** — 처음 링크를 클릭한 시각

이 데이터를 활용하여:
- 후속 조치가 필요한 활발한 참여 사용자를 찾을 수 있습니다.
- 목록 정리를 위해 바운스된 주소를 식별할 수 있습니다.
- 특정 주소의 발송 문제를 디버깅할 수 있습니다.

## 다음 단계
- 분석 결과를 바탕으로 향후 캠페인의 제목, 톤, 발송 시점 등을 개선하세요.
- 도달률 보호를 위해 바운스된 연락처를 목록에서 정리하세요.
- 요금제 한도에 가까워진 경우 [결제 안내](/ko/billing/overview)에서 전송 한도를 확인하세요.
