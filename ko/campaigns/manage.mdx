## 이 페이지의 도움말
초안 상태의 캠페인을 편집하고, 발송 또는 예약하며, 진행 중인 캠페인을 중단하는 방법을 설명합니다. 또한 캠페인이 발송된 후 수정할 수 있는 항목과 수정 불가능한 항목을 안내합니다.

## 시작하기 전에
- 최소 하나 이상의 캠페인(초안 또는 발송 완료)이 있어야 합니다.
- 편집 권한이 있는 소유자(Owner) 또는 팀 멤버여야 합니다.

## 캠페인 편집하기
**초안(Draft) 상태의 캠페인만 편집할 수 있습니다.** 캠페인이 발송된 후에는 콘텐츠와 오디언스가 고정되어 수정할 수 없습니다.

초안 상태에서는 다음 항목을 변경할 수 있습니다:

- 오디언스 (전체 연락처 또는 태그)
- 브랜드 설정
- 캠페인 목표 및 AI 생성 콘텐츠
- 예약 일정 (날짜 및 시간)
- 자동 중단 규칙 (드립 시퀀스의 경우)

1. **캠페인(Campaigns)** 메뉴로 이동합니다.
2. 캠페인을 클릭하거나 **편집(Edit)**을 누릅니다.
3. 마법사 단계에서 내용을 수정합니다.
4. 저장합니다. 이미 예약된 캠페인이었다면 새로운 일정이 적용됩니다.

## 즉시 발송하기
1. 초안 상태의 캠페인을 엽니다.
2. **지금 발송(Send now)**을 클릭합니다.
3. 확인합니다. 이메일이 발송 대기열에 추가되며 수 분 내에 발송이 시작됩니다.

<Note>
발송은 취소할 수 없습니다. 승인 전 오디언스와 콘텐츠를 다시 한번 꼭 확인해 주세요.
</Note>

## 나중에 발송하도록 예약하기
1. 초안 상태의 캠페인을 엽니다.
2. **예약(Schedule)** 버튼을 선택하고 원하는 날짜와 시간을 지정합니다.
3. 저장합니다. 캠페인 상태가 **예약됨(Scheduled)**으로 변경됩니다.
4. 설정된 시간에 이메일이 자동으로 발송됩니다.

실행 전까지는 예약된 캠페인을 편집하거나 취소할 수 있습니다.

## 캠페인 중단하기
발송을 즉시 멈추고 싶을 때 **캠페인 중단(Stop campaign)** 기능을 사용하세요. 다음 상황에 적용됩니다:

- 현재 **발송 중(Sending)**인 캠페인
- **예약된(Scheduled)** 캠페인 (발송 취소)
- **드립 시퀀스(Drip sequences)** — 아직 이메일을 받지 않은 수신자에 대한 이후 단계 발송 중단

이미 발송된 이메일은 회수할 수 없습니다. 중단 전에 메일을 받은 수신자의 편지함에는 메시지가 유지됩니다.

## 초안 vs 발송 완료: 변경 가능 여부

| 활동 | 초안 (Draft) | 예약됨 (Scheduled) | 발송 완료 (Sent) |
| :--- | :--- | :--- | :--- |
| 콘텐츠 편집 | 예 | 예 | 아니요 |
| 오디언스 변경 | 예 | 예 | 아니요 |
| 일정 변경 | 예 | 예 | 해당 없음 |
| 즉시 발송 | 예 | 예 | 해당 없음 |
| 중단 (Stop) | 해당 없음 | 예 | 예* |
| 분석 데이터 확인 | 아니요 | 아니요 | 예 |

\*발송 완료된 캠페인을 중단하면 이후의 단계(예: 드립 시퀀스) 발송만 방지합니다. 이미 발송된 이전 메일에는 영향을 주지 않습니다.
