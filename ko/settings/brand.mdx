## 이 페이지의 도움말
브랜드의 전역 정체성을 설정하는 방법을 안내합니다. 이 정보는 이메일의 발신자(From) 헤더에 사용되며, AI가 더 정확한 내용을 생성할 수 있도록 브랜드 컨텍스트를 제공합니다. 또한 캠페인 복사본의 기본 언어를 설정할 수 있습니다.

## 시작하기 전에
- Nudgen 계정
- 소유자(Owner) 또는 관리자(Admin) 권한
- 브랜드 자산 (이름, 로고 및 발신용 이메일)

## 브랜드 정체성 설정
대시보드의 **설정(Settings)** → **브랜드(Brand)** 메뉴에서 다음 옵션에 접근할 수 있습니다:

| 옵션 | 설명 |
| :--- | :--- |
| **Brand name (브랜드명)** | 회사 또는 제품 이름(예: "Nudgen"). AI 생성 시 컨텍스트로 사용됩니다. |
| **Brand email (브랜드 이메일)** | 캠페인의 기본 회신용 이메일 주소(예: `support@yourbrand.com`). |
| **Brand description (브랜드 설명)** | 브랜드가 무엇을 하는지 보여주는 짧은 요약입니다. AI가 더 나은 문구를 작성하는 데 도움을 줍니다. |
| **Email language (이메일 언어)** | 제목과 본문을 생성할 때 AI가 사용하는 기본 언어입니다. |
| **Logo (로고)** | 브랜드의 로고(PNG 또는 JPEG, 최대 5MB). 이메일 템플릿 및 미리보기에 표시됩니다. |

## 지원되는 이메일 언어
Nudgen은 AI 생성 이메일 문구를 위해 다양한 언어를 지원합니다. **설정** → **브랜드**에서 기본 언어를 설정하거나, 캠페인별로 변경할 수 있습니다.

현재 지원되는 언어:

| 언어 | 원어 표기 | 코드 |
| :--- | :--- | :--- |
| **English** | English | `en` |
| **Spanish** | Español | `es` |
| **French** | Français | `fr` |
| **Chinese** | 中文 | `zh` |
| **Japanese** | 日本語 | `ja` |
| **Vietnamese** | Tiếng Việt | `vi` |
| **Korean** | 한국어 | `ko` |
| **German** | Deutsch | `de` |
| **Portuguese** | Português | `pt` |
| **Italian** | Italiano | `it` |
| **Hindi** | हिन्दी | `hi` |
| **Indonesian** | Bahasa Indonesia | `id` |
| **Urdu** | اردو | `ur` |
| **Turkish** | Türkçe | `tr` |
| **Arabic** | العربية | `ar` |
| **Russian** | Русский | `ru` |

<Tip>
타겟팅하려는 언어가 목록에 없는 경우 지원팀에 문의해 주세요. 더 많은 지역 언어 지원을 지속적으로 추가하고 있습니다.
</Tip>
