## 도움말 범위
주요 영역별로 분류된 문제와 해결 방법을 안내합니다. 해결하려는 문제가 목록에 없는 경우 아래의 상세 가이드를 확인하거나 지원팀에 문의해 주세요.

## 발송 및 도달률 관련

| 문제 상황 | 해결 가이드가 있는 곳 |
| :--- | :--- |
| 이메일이 발송되지 않음 / 캠페인이 멈춰 있음 | [발송 및 도달률](/ko/troubleshooting/deliverability#이메일이-발송되지-않음) |
| 바운스율 또는 스팸 신고율이 높음 | [발송 및 도달률](/ko/troubleshooting/deliverability#바운스-및-신고-관리) |
| 테스트 이메일이 도착하지 않음 | 스팸함을 확인하고, 설정된 계정 이메일 주소가 맞는지 확인하세요. [캠페인 생성](/ko/campaigns/create) 문제 해결 섹션을 참조하세요. |
| 오픈 수 또는 클릭 수가 업데이트되지 않음 | [발송 및 도달률](/ko/troubleshooting/deliverability#추적-및-지표) |

## 연락처 및 가져오기 관련

| 문제 상황 | 해결 가이드가 있는 곳 |
| :--- | :--- |
| 업로드 실패 또는 일부 행이 생략됨 | [연락처 업로드](/ko/contacts/upload#문제-해결) — CSV 형식, 이메일 컬럼, 요금제 한도를 확인하세요. |
| 연락처 한도 초과 | 요금제를 업그레이드하거나 기존 연락처를 삭제하세요. [결제 안내](/ko/billing/overview). |
| 예상 도달 범위가 0으로 나옴 | 연락처를 추가하거나 가져오세요. 연락처가 활성 상태(수신 거부/차단되지 않음)인지 확인하세요. [연락처 개요](/ko/contacts/overview). |
| 연락처 정보 확인 불가 / 마스킹됨 | 해당 워크스페이스에 대한 접근 권한이 필요합니다. 확인(Reveal)은 세션 단위이며 새로고침 시 다시 마스킹됩니다. [연락처 개요](/ko/contacts/overview). |

## 권한 및 접근 관련

| 문제 상황 | 해결 가이드가 있는 곳 |
| :--- | :--- |
| 초대장 이메일을 받지 못함 | [권한 및 접근](/ko/troubleshooting/access#초대-관련) — 스팸함 확인, 재발송, 이메일 주소 확인을 진행하세요. |
| 워크스페이스가 잘못되었거나 데이터를 볼 수 없음 | 사이드바의 팀 전환기(switcher)를 사용하세요. [워크스페이스 및 역할](/ko/concepts/workspaces-and-roles). |
| 멤버 초대 또는 역할 변경을 할 수 없음 | 소유자(Owner)와 관리자(Admin)만 가능합니다. [팀 설정](/ko/settings/team). |
| 멤버 해제 후 접근 권한 상실 | 해당 워크스페이스 소유자에게 문의하세요. 결제 관리는 소유자만 가능합니다. |

  상세 가이드: 초대, 역할, 워크스페이스 접근 권한.
</Card>

## 결제 및 구독 관련

| 문제 상황 | 해결 가이드가 있는 곳 |
| :--- | :--- |
| 연락처 또는 이메일 한도 도달 | [결제 문제 해결](/ko/troubleshooting/billing#한도-관련) — 업그레이드하거나 초기화 시점까지 기다리세요. |
| 결제 실패 / 카드 승인 거절 | [결제 문제 해결](/ko/troubleshooting/billing#결제-관련) — 포털에서 카드 정보를 업데이트하세요. |
| Trial 종료 / 캠페인 발송 불가 | 유료 요금제로 업그레이드하세요. [결제 개요](/ko/billing/overview). |
| 요금제 변경 또는 취소 | 구독 관리(Manage subscription) 포털을 이용하세요. [결제 개요](/ko/billing/overview). |

<Card title="결제 및 구독" icon="credit-card" href="/ko/troubleshooting/billing">
  상세 가이드: 한도, 결제, Trial, 포털 이용 방법.
</Card>

## 캠페인 및 콘텐츠 관련

| 문제 상황 | 해결 가이드가 있는 곳 |
| :--- | :--- |
| AI 생성 문구가 브랜드 톤과 다름 | 브랜드 설정(맞춤형 프롬프트, 예시 이메일)을 보강하세요. [브랜드 개요](/ko/brand/overview). |
| 캠페인을 수정할 수 없음 | 초안(Draft) 및 예약된(Scheduled) 캠페인만 수정 가능합니다. [캠페인 관리](/ko/campaigns/manage). |
| 캠페인을 중단하고 싶음 | 목록이나 상세 페이지에서 **캠페인 중단(Stop campaign)**을 사용하세요. [캠페인 관리](/ko/campaigns/manage). |
| 서명이 누락되었거나 잘못됨 | 캠페인 마법사에서 서명을 선택하고 미리보기로 확인하세요. [서명 설정](/ko/settings/signatures). |
