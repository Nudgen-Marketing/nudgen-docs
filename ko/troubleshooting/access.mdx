## 초대 관련

**초대받은 사람이 초대 이메일을 받지 못했습니다**

- 수신자에게 **스팸/정크 메일함**을 확인하고 발신자 주소를 주소록에 추가하거나 안전한 발신자로 지정해 달라고 요청하세요.
- **이메일 주소**가 맞는지 확인하세요. **설정(Settings)** → **팀(Team)** 메뉴에서 대기 중인 초대를 다시 발송해 보세요.
- 초대가 **만료**되었다면 해당 이메일로 새 초대를 생성하세요. 만료된 링크는 다시 사용할 수 없습니다.

**초대를 수락했는데 워크스페이스가 보이지 않습니다**

- 사이드바(헤더)에 있는 **팀 전환기(team switcher)**를 선택해 보세요. 새로운 워크스페이스가 목록에 나타날 것입니다. 해당 워크스페이스를 골라 전환하세요. [워크스페이스 및 역할](/ko/concepts/workspaces-and-roles)을 참조하세요.

**초대 링크가 만료되었거나 유효하지 않다고 나옵니다**

- 초대 링크는 일정 기간이 지나면 만료됩니다. 워크스페이스 소유자나 관리자에게 **새로운 초대**를 요청하세요. 이전 링크는 더 이상 사용할 수 없습니다.

## 역할 및 권한 관련

**멤버를 초대하거나 역할을 변경할 수 없습니다**

- **소유자(Owner)**와 **관리자(Admin)**만 멤버 초대 및 역할 변경이 가능합니다. **멤버(Member)**는 팀 목록을 볼 수는 있지만 수정할 수는 없습니다. 관리 권한이 필요한 경우 소유자나 다른 관리자에게 역할 변경을 요청하세요. [팀 설정](/ko/settings/team)을 참조하세요.

**결제 메뉴에 접근하거나 요금제를 변경할 수 없습니다**

- **결제(Billing)** 관리(업그레이드, 결제 수단, 취소 등)는 **소유자(Owner)**만 가능합니다. 관리자와 멤버는 사용량을 볼 수만 있으며 구독을 관리할 수는 없습니다. 소유자에게 업그레이드나 결제 정보 업데이트를 요청하세요.

**멤버를 삭제했는데 다시 추가할 수 있나요?**

- 소유자나 관리자가 **초대장**을 새로 발송해야 합니다. 삭제된 멤버의 이전 권한은 취소된 상태입니다.

## 워크스페이스 선택

**잘못된 데이터(엉뚱한 캠페인이나 연락처)가 보입니다**

- 모든 데이터는 **워크스페이스별**로 나뉩니다. 사이드바의 **팀 전환기**를 사용하여 올바른 워크스페이스에 있는지 확인하세요. 워크스페이스마다 각기 다른 연락처, 캠페인, 브랜드 설정 및 결제 정보가 존재합니다.

**여러 개의 워크스페이스가 있는데 어떻게 전환하나요?**

- 사이드바 헤더에 있는 현재 워크스페이스 이름을 클릭하여 팀 전환기를 엽니다. 원하는 워크스페이스를 선택하면 페이지 내용이 해당 워크스페이스의 데이터로 업데이트됩니다.

**초대받은 워크스페이스가 보이지 않습니다**

- 이메일의 링크를 클릭하여 초대를 **수락**했는지 확인하세요. 이미 수락했다면 로그아웃 후 다시 로그인해 보거나, 새로고침한 뒤 팀 전환기를 다시 확인해 보세요. 여전히 나타나지 않는다면 초대가 취소되었을 수 있으니 새로 요청하시기 바랍니다.
