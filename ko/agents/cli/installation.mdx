## 빠른 설치

Nudgen CLI를 설치하는 가장 쉬운 방법은 원라이너 설치 스크립트를 사용하는 것입니다.

### macOS & Linux

터미널에서 다음 명령을 실행하세요:

```bash
curl -fsSL https://raw.githubusercontent.com/Nudgen-Marketing/nudgen-cli/main/scripts/install.sh | bash
```

### Go를 통한 설치

Go 1.22+가 설치되어 있다면 CLI를 직접 설치할 수 있습니다:

```bash
go install github.com/Nudgen-Marketing/nudgen-cli@latest
```

### 소스에서 빌드

Go 1.22+가 설치되어 있다면 소스에서 직접 빌드할 수 있습니다:

```bash
git clone https://github.com/Nudgen-Marketing/nudgen-cli.git
cd nudgen-cli
make build
```

바이너리는 `./bin/nudgen` 위치에 생성됩니다.

## 초기 설정

설치가 완료되면 버전을 확인하여 설치를 검증합니다:

```bash
nudgen version
```

### 인증

CLI 사용을 시작하려면 개인 액세스 토큰(PAT)으로 인증해야 합니다.

1. 로그인 명령을 실행합니다:
   ```bash
   nudgen login
   ```
2. 대화형 프롬프트에 따라 토큰을 입력합니다.
3. 자신의 신원을 확인합니다:
   ```bash
   nudgen whoami
   ```

### 업데이트

최신 버전으로 업데이트하려면 설치 스크립트를 다시 실행하거나 내장된 업데이트 명령을 사용하세요(가능한 경우):

```bash
nudgen update
```

## 보안
