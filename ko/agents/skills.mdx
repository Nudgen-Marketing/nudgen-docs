## 에이전트에 기능 추가

Nudgen 스킬은 Claude Code, Codex, Cursor와 같은 코딩 에이전트를 위해 설계된 모듈식 지침 및 스크립트입니다. 이는 마케팅 워크플로우 및 Nudgen 통합 패턴에 대한 에이전트의 이해를 확장합니다.

## 사용 가능한 스킬

### CLI 스킬
Nudgen CLI 설치 및 인증, 팀, 연락처, 캠페인, 브랜드 아이덴티티 및 추천 트래킹을 관리합니다.

```bash
npx skills add https://github.com/Nudgen-Marketing/skills --global --skill cli
```

**상세 정보**: 터미널에서 설치, PAT를 통한 인증, 연락처, 캠페인, 브랜드 아이덴티티 및 추천 데이터를 관리합니다. 멀티 팀 컨텍스트 전환 및 스크립팅을 위한 `--json` 출력을 지원합니다.

---

### API 스킬
연락처 CRUD, 캠페인, 팀, 브랜드 구성, 통계, 어필리에이트 데이터, SDK 패턴, 속도 제한(rate limits) 및 멱등성 키(idempotency keys)를 다룹니다.

```bash
npx skills add https://github.com/Nudgen-Marketing/skills --global --skill api
```

**상세 정보**: 연락처, 캠페인, 템플릿, 브랜드 설정, 팀 관리, SDK vs REST API 패턴, 속도 제한, 멱등성 키 및 오류 처리를 포함합니다. 인증에는 `Authorization: Bearer <PAT>`를 사용합니다.

---

### 이메일 모범 사례 스킬
범용 이메일 스킬입니다. 전달성, 구조 및 라이프사이클 패턴을 감사하며 Nudgen 사용 여부와 관계없이 작동합니다.

```bash
npx skills add https://github.com/Nudgen-Marketing/skills --global --skill email-sending-best-practices
```

**상세 정보**: 발신자 인증(SPF, DKIM, DMARC)을 검증하고, 트랜잭션 이메일과 마케팅 이메일의 분리 여부를 확인하며, 수신 거부 처리를 검토하고, 누락된 라이프사이클 단계를 표시하며, 스팸 트리거에 대한 콘텐츠 점수를 산출합니다.

## 기여하기

저희 스킬 라이브러리에 대한 커뮤니티의 기여를 환영합니다. Nudgen 또는 일반적인 마케팅 자동화를 위한 유용한 스킬을 만드셨다면 [GitHub 리포지토리](https://github.com/Nudgen-Marketing/skills)에 PR을 제출해 주세요.
