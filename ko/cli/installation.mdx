## 빠른 설치

가장 쉬운 설치 방법은 한 줄 설치 스크립트를 사용하는 것입니다.

### macOS & Linux

터미널에서 다음 명령을 실행하세요:

```bash
curl -fsSL https://raw.githubusercontent.com/Nudgen-Marketing/nudgen-cli/main/scripts/install.sh | bash
```

### 소스에서 빌드

Go 1.22+가 설치되어 있는 경우 소스에서 빌드할 수 있습니다:

```bash
git clone https://github.com/Nudgen-Marketing/nudgen-cli.git
cd nudgen-cli
make build
```

바이너리는 `./bin/nudgen`에 생성됩니다.

## 초기 설정

설치 후 버전을 확인하여 정상적으로 설치되었는지 확인합니다:

```bash
nudgen version
```

### 인증

CLI 사용을 시작하려면 개인 액세스 토큰(PAT)으로 인증해야 합니다.

1. 로그인 명령을 실행합니다:
   ```bash
   nudgen login
   ```
2. 대화형 프롬프트에 따라 토큰을 입력합니다.
3. 신원을 확인합니다:
   ```bash
   nudgen whoami
   ```

### 업데이트

최신 버전으로 업데이트하려면 설치 스크립트를 다시 실행하거나 내장된 업데이트 명령을 사용하세요:

```bash
nudgen update
```

## 보안

Nudgen CLI는 시스템 키체인을 사용하여 PAT를 안전하게 저장합니다. 명령 출력에 원본 토큰을 표시하거나 로그에 기록하지 않습니다.
