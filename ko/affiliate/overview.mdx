## 운영 방식

Nudgen과 함께 수동 수입을 시작하는 세 가지 간단한 단계.

1. **고유 링크 받기** — 파트너로 가입하고 30일 동안 모든 클릭과 전환을 추적하는 개인 추천 링크를 받으세요.
2. **자유로운 홍보** — 어디에서나 링크를 공유하세요. 유료 광고, SNS, 블로그 또는 이메일을 허용합니다. 완전한 마케팅 자유를 드립니다.
3. **평생 수동 수입** — 추천 고객이 구독을 갱신할 때마다 10% 반복 커미션을 받게 됩니다. 평생 동안, 한도 없이 제공됩니다.

## 커미션 구조

모든 유료 플랜에 대해 커미션이 반복적으로 발생합니다. 고객을 더 많이 유치할수록 귀하의 커미션도 커집니다.

| 항목 | 상세 내용 |
| :--- | :--- |
| **커미션 비율** | 10% 반복 |
| **쿠키 지속 기간** | 30일 |
| **최소 지급 금액** | $50 USD |
| **결제 방법** | PayPal |
| **승인 절차** | 즉시 / 별도 승인 불필요 |
