## Hızlı Kurulum

Nudgen CLI'yı kurmanın en kolay yolu, tek satırlık kurulum betiğimizi kullanmaktır.

### macOS ve Linux

Terminalinizde aşağıdaki komutu çalıştırın:

```bash
curl -fsSL https://raw.githubusercontent.com/Nudgen-Marketing/nudgen-cli/main/scripts/install.sh | bash
```

### Kaynaktan Kurulum

Go 1.22+ kuruluysa, kaynaktan derleyebilirsiniz:

```bash
git clone https://github.com/Nudgen-Marketing/nudgen-cli.git
cd nudgen-cli
make build
```

Binary dosyası `./bin/nudgen` altında oluşturulacaktır.

## İlk Yapılandırma

Kurulumdan sonra, her şeyin doğru kurulduğundan emin olmak için sürümü kontrol edin:

```bash
nudgen version
```

### Kimlik Doğrulama

CLI kullanmaya başlamak için bir Kişisel Erişim Anahtarı (PAT) ile kimlik doğrulaması yapmanız gerekir.

1. Giriş komutunu çalıştırın:
   ```bash
   nudgen login
   ```
2. İnteraktif istemleri takip ederek anahtarınızı girin.
3. Kimliğinizi doğrulayın:
   ```bash
   nudgen whoami
   ```

### Güncelleme

En son sürüme güncellemek için kurulum betiğini tekrar çalıştırmanız veya yerleşik güncelleme komutunu kullanmanız yeterlidir:

```bash
nudgen update
```

## Güvenlik

Nudgen CLI, PAT'inizi güvenli bir şekilde saklamak için sistem anahtar deposunu (keychain) kullanır. Komut çıktılarında veya günlüklerde asla ham anahtarlarınızı görüntülemeyiz.

Sıradaki adımda, [temel komutları](/tr/cli/commands) keşfedin.
