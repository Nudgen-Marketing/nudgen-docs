---
title: "Nhà phát triển và API"
description: "Khả năng tích hợp và mức độ sẵn có của API"
---

Hiện tại Nudgen chưa cung cấp public API, webhook hay SDK nhúng. Mọi thao tác đều thực hiện qua ứng dụng web tại [nudgen.net](https://nudgen.net). Nếu bạn cần tích hợp hoặc truy cập theo cách lập trình, hãy liên hệ với chúng tôi. Chúng tôi có thể bổ sung API trong các phiên bản tới.
