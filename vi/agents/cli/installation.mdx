## Cài đặt nhanh

Cách dễ nhất để cài đặt Nudgen CLI là sử dụng tập lệnh cài đặt một dòng của chúng tôi.

### macOS & Linux

Chạy lệnh sau trong terminal của bạn:

```bash
curl -fsSL https://raw.githubusercontent.com/Nudgen-Marketing/nudgen-cli/main/scripts/install.sh | bash
```

### Qua Go

Nếu bạn đã cài đặt Go 1.22+, bạn có thể cài đặt CLI trực tiếp:

```bash
go install github.com/Nudgen-Marketing/nudgen-cli@latest
```

### Từ mã nguồn (Source)

Nếu bạn đã cài đặt Go 1.22+, bạn có thể build từ mã nguồn:

```bash
git clone https://github.com/Nudgen-Marketing/nudgen-cli.git
cd nudgen-cli
make build
```

Tệp thực thi sẽ có sẵn tại `./bin/nudgen`.

## Thiết lập ban đầu

Sau khi cài đặt, hãy xác nhận bằng cách kiểm tra phiên bản:

```bash
nudgen version
```

### Xác thực

Để bắt đầu sử dụng CLI, bạn cần xác thực bằng Personal Access Token (PAT).

1. Chạy lệnh đăng nhập:
   ```bash
   nudgen login
   ```
2. Làm theo các hướng dẫn tương tác để nhập token của bạn.
3. Xác minh danh tính của bạn:
   ```bash
   nudgen whoami
   ```

### Cập nhật

Để cập nhật lên phiên bản mới nhất, chỉ cần chạy lại tập lệnh cài đặt hoặc sử dụng lệnh cập nhật tích hợp (nếu có):

```bash
nudgen update
```

## Bảo mật

Nudgen CLI sử dụng hệ thống keychain của máy tính để lưu trữ PAT của bạn một cách an toàn. Chúng tôi không bao giờ ghi nhật ký hoặc xuất token thô trong bất kỳ kết quả lệnh nào.

Tiếp theo, hãy khám phá [các lệnh cốt lõi](/vi/agents/cli/commands).
