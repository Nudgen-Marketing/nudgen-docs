## نظرة عامة

تعد واجهة السطر البرمجي لـ Nudgen (CLI) أداة قوية مصممة لكل من البشر ووكلاء الذكاء الاصطناعي. وهي توفر طريقة سلسة لإدارة الحملات وجهات الاتصال وهويات العلامة التجارية وإحصائيات الإحالة مباشرة من جهازك الطرفي (Terminal).

سواء كنت مطوراً يتطلع إلى أتمتة تدفقات عمل التسويق الخاصة بك أو وكيل ذكاء اصطناعي يحتاج إلى تفاعل عالي الجودة مع منصة Nudgen، فإن CLI هي واجهتك الأساسية.

## الغرض

تخدم CLI أدواراً متعددة:

1. **للبشر**: توفر طريقة سريعة لسرد الحملات واستيراد جهات الاتصال والتحقق من أرباح الإحالة دون فتح متصفح الويب.
2. **لوكلاء الذكاء الاصطناعي**: توفر واجهة قابلة للتفسير آلياً لأتمتة عمليات التسويق المعقدة وإدارة العلامات التجارية.
3. **للمطورين**: أدوات مساعدة للتطوير المحلي وإدارة المستودعات داخل نظام Nudgen البيئي.

## أنماط التفاعل

تتبع CLI مبدأ "التفاعل عبر واجهة البرمجة (API)". وهي تستخدم استدعاءات RESTful API القياسية للتواصل مع خادم Nudgen الخلفي مع إدارة حالة الجلسة محلياً بشكل آمن.

- **معيار JSON**: تدعم جميع الأوامر خيار `--json`. هذا أمر بالغ الأهمية للتواصل بين وكيل وآخر.
- **مراعاة السياق**: يتم حصر العمليات مثل سرد الحملات تلقائياً ضمن نطاق فريقك النشط.

بعد ذلك، تعرف على كيفية [تثبيت CLI](/ar/agents/cli/installation).
