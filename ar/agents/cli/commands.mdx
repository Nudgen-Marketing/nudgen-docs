## نظرة عامة

استكشف الأوامر الأساسية المتاحة في Nudgen CLI. يتم تجميع معظم الأوامر حسب نوع المورد (الحملات، جهات الاتصال، الفرق، إلخ).

### المصادقة والهوية

| الأمر | الوصف |
| :--- | :--- |
| `nudgen login` | المصادقة باستخدام رمز الوصول الشخصي (PAT). |
| `nudgen whoami` | إظهار مالك الرمز الحالي ومعلومات المستخدم. |
| `nudgen logout` | مسح بيانات الاعتماد المحلية. |

### إدارة الفريق

| الأمر | الوصف |
| :--- | :--- |
| `nudgen teams list` | سرد جميع الفرق التي تنتمي إليها. |
| `nudgen teams current` | إظهار سياق الفريق النشط حالياً. |
| `nudgen teams switch <id>` | التبديل إلى فريق معين. |

### الحملات

إدارة جهود التسويق ضمن الفريق النشط.

| الأمر | الوصف |
| :--- | :--- |
| `nudgen campaigns list` | سرد الحملات النشطة في الفريق الحالي. |
| `nudgen campaigns create` | معالج إنشاء تفاعلي. |
| `nudgen campaigns update <id>` | تعديل إعدادات الحملة الموجودة. |
| `nudgen campaigns delete <id>` | إزالة سجل الحملة. |

### جهات الاتصال (إدارة العملاء المحتملين)

| الأمر | الوصف |
| :--- | :--- |
| `nudgen contacts list` | سرد جميع العملاء المحتملين في الفريق الحالي. |
| `nudgen contacts create` | إضافة سجل عميل محتمل جديد يدوياً. |
| `nudgen contacts import` | مخطط CSV تفاعلي للاستيراد الجماعي. |
| `nudgen contacts update <id>` | تحرير تفاصيل العميل المحتمل. |
| `nudgen contacts delete <id>` | إزالة سجل جهة الاتصال. |

### هويات العلامة التجارية

| الأمر | الوصف |
| :--- | :--- |
| `nudgen brand list` | سرد جميع أصوات العلامة التجارية للفريق. |
| `nudgen brand create` | معالج لتهيئة صوت علامة تجارية جديد. |
| `nudgen brand update <id>` | تحرير تكوينات العلامة التجارية. |
| `nudgen brand delete <id>` | إزالة هوية العلامة التجارية. |

### الإحالات والأرباح

| الأمر | الوصف |
| :--- | :--- |
| `nudgen referral stats` | عرض ملخص عالي المستوى لأداء الإحالة. |
| `nudgen referral link` | احصل على رابط الإحالة الفريد الخاص بك. |
| `nudgen referral activity` | عرض سجل النقرات والتسجيلات الأخيرة. |
| `nudgen referral referrals` | سرد الفرق التي تمت إحالتها بواسطتك وحالتها. |
| `nudgen referral payouts` | عرض سجل طلبات الدفع وحالتها. |

بعد ذلك، تعرف على كيفية استخدام هذه الأوامر لـ [وكلاء الذكاء الاصطناعي](/ar/agents/cli/agentic-usage).
