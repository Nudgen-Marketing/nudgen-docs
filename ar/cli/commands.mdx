## نظرة عامة

تعرف على الأوامر الأساسية المتاحة في CLI لـ Nudgen. معظم الأوامر مجمعة حسب نوع المورد (الحملات، جهات الاتصال، الفرق، إلخ).

### المصادقة والهوية

| الأمر | الوصف |
| :--- | :--- |
| `nudgen login` | المصادقة باستخدام رمز الوصول الشخصي (PAT). |
| `nudgen whoami` | عرض صاحب الرمز الحالي ومعلومات المستخدم. |
| `nudgen logout` | مسح بيانات الاعتماد المحلية. |

### إدارة الفرق

| الأمر | الوصف |
| :--- | :--- |
| `nudgen teams list` | سرد جميع الفرق التي تنتمي إليها. |
| `nudgen teams current` | عرض سياق الفريق النشط حاليًا. |
| `nudgen teams switch <id>` | الانتقال إلى فريق محدد. |

### الحملات (Campaigns)

إدارة جهودك التسويقية ضمن الفريق النشط.

| الأمر | الوصف |
| :--- | :--- |
| `nudgen campaigns list` | سرد الحملات النشطة في الفريق الحالي. |
| `nudgen campaigns create` | معالج إنشاء تفاعلي. |
| `nudgen campaigns update <id>` | تعديل إعدادات حملة موجودة. |
| `nudgen campaigns delete <id>` | حذف سجل حملة. |

### جهات الاتصال (إدارة العملاء المحتملين)

| الأمر | الوصف |
| :--- | :--- |
| `nudgen contacts list` | سرد جميع العملاء المحتملين في الفريق الحالي. |
| `nudgen contacts create` | إضافة سجل عميل محتمل يدويًا. |
| `nudgen contacts import` | مخطط CSV تفاعلي للاستيراد الجماعي. |
| `nudgen contacts update <id>` | تحرير تفاصيل العميل المحتمل. |
| `nudgen contacts delete <id>` | حذف سجل جهة اتصال. |

### هويات العلامة التجارية (Brand)

| الأمر | الوصف |
| :--- | :--- |
| `nudgen brand list` | سرد جميع نبرات العلامات التجارية للفريق. |
| `nudgen brand create` | معالج لتهيئة نبرة علامة تجارية جديدة. |
| `nudgen brand update <id>` | تحرير تكوينات العلامة التجارية. |
| `nudgen brand delete <id>` | حذف هوية علامة تجارية. |

### الإحالات والأرباح

| الأمر | الوصف |
| :--- | :--- |
| `nudgen referral stats` | عرض ملخص الأداء لبرنامج الإحالة. |
| `nudgen referral link` | الحصول على رابط الإحالة الفريد الخاص بك. |
| `nudgen referral activity` | عرض سجل النقرات والتسجيلات الأخيرة. |
| `nudgen referral referrals` | سرد الفرق التي تمت إحالتها وحالتها. |
| `nudgen referral payouts` | عرض تاريخ وحالة طلبات الدفع. |

بعد ذلك، تعرف على كيفية استخدام هذه الأوامر لـ [وكلاء الذكاء الاصطناعي](/ar/cli/agentic-usage).
