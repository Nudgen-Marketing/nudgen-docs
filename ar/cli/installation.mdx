## التثبيت السريع

أسهل طريقة لتثبيت CLI الخاص بـ Nudgen هي استخدام نص التثبيت البرمجي ذو السطر الواحد.

### macOS و Linux

قم بتشغيل الأمر التالي في جهازك الطرفي (Terminal):

```bash
curl -fsSL https://raw.githubusercontent.com/Nudgen-Marketing/nudgen-cli/main/scripts/install.sh | bash
```

### البناء من المصدر

إذا كان لديك Go 1.22+ مثبتًا، يمكنك البناء من المصدر مباشرة:

```bash
git clone https://github.com/Nudgen-Marketing/nudgen-cli.git
cd nudgen-cli
make build
```

سيتم إنشاء الملف التنفيذي في المسار `./bin/nudgen`.

## الإعداد الأولي

بعد التثبيت، قم بالتحقق من الإصدار للتأكد من نجاح التثبيت:

```bash
nudgen version
```

### المصادقة

لبدء استخدام الـ CLI، يجب عليك المصادقة باستخدام رمز الوصول الشخصي (PAT).

1. قم بتشغيل أمر تسجيل الدخول:
   ```bash
   nudgen login
   ```
2. اتبع المطالبات التفاعلية لإدخال الرمز الخاص بك.
3. تحقق من هويتك:
   ```bash
   nudgen whoami
   ```

### التحديث

للتحديث إلى أحدث إصدار، ببساطة قم بتشغيل نص التثبيت البرمجي مرة أخرى أو استخدم أمر التحديث المدمج:

```bash
nudgen update
```

## الأمان

تستخدم CLI الخاصة بـ Nudgen سلسلة مفاتيح النظام (Keychain) لتخزين رمز الوصول الشخصي بأمان. نحن لا نقوم أبدًا بعرض الرموز الخام في مخرجات الأوامر أو السجلات.

بعد ذلك، استكشف [الأوامر الأساسية](/ar/cli/commands).
