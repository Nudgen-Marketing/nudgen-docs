## نظرة عامة

تعد واجهة السطر البرمجي (CLI) لـ Nudgen أداة قوية مصممة لكل من البشر ووكلاء الذكاء الاصطناعي (AI Agents). توفر طريقة سلسة لإدارة الحملات، جهات الاتصال، هويات العلامات التجارية، وإحصائيات الإحالة مباشرة من جهازك الطرفي (Terminal).

سواء كنت مطورًا يتطلع إلى أتمتة سير عمل التسويق الخاص بك أو وكيل ذكاء اصطناعي يحتاج إلى تفاعل عالي الجودة مع منصة Nudgen، فإن الـ CLI هي واجهتك الأساسية.

## الغرض

تؤدي واجهة السطر البرمجي أدوارًا متعددة:

1. **للمستخدمين البشر**: سرد الحملات بسرعة، استيراد جهات الاتصال، والتحقق من الأرباح دون الحاجة لفتح المتصفح.
2. **لوكلاء الذكاء الاصطناعي**: توفر واجهة قابلة للتفسير آليًا لأتمتة عمليات التسويق المعقدة وإدارة العلامات التجارية.
3. **للمطورين**: أدوات مساعدة للتطوير المحلي وإدارة المستودعات داخل بيئة Nudgen.

## أنماط التفاعل

تتبع الـ CLI مبدأ "التفاعل عبر واجهة البرمجة (API)". وتستخدم طلبات RESTful API القياسية للتواصل مع خادم Nudgen مع إدارة حالة الجلسة محليًا بشكل آمن.

- **معيار JSON**: تدعم جميع الأوامر خيار `--json` الضروري للتواصل بين الوكلاء.
- **إدراك السياق**: يتم حصر العمليات (مثل سرد الحملات) تلقائيًا ضمن فريقك النشط حاليًا.
