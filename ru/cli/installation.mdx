## Быстрая установка

Самый простой способ установить Nudgen CLI — использовать скрипт установки одной командой.

### macOS и Linux

Выполните эту команду в терминале:

```bash
curl -fsSL https://raw.githubusercontent.com/Nudgen-Marketing/nudgen-cli/main/scripts/install.sh | bash
```

### Сборка из исходников

Если у вас установлен Go 1.22+, вы можете собрать проект самостоятельно:

```bash
git clone https://github.com/Nudgen-Marketing/nudgen-cli.git
cd nudgen-cli
make build
```

Бинарный файл будет создан в папке `./bin/nudgen`.

## Первоначальная настройка

После установки проверьте работоспособность, посмотрев версию:

```bash
nudgen version
```

### Авторизация

Для начала работы необходимо авторизоваться с помощью Personal Access Token (PAT).

1. Запустите команду входа:
   ```bash
   nudgen login
   ```
2. Следуйте инструкциям и введите ваш токен.
3. Проверьте статус авторизации:
   ```bash
   nudgen whoami
   ```

### Обновление

Чтобы обновиться до последней версии, просто запустите скрипт установки еще раз или используйте встроенную команду:

```bash
nudgen update
```

## Безопасность

Nudgen CLI использует системную связку ключей (Keychain) для безопасного хранения вашего токена. Мы никогда не выводим и не логгируем «сырые» токены.
