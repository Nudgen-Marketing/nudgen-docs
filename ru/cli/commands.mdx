## Обзор

Ознакомьтесь с основными командами Nudgen CLI. Большинство команд сгруппированы по типу ресурса (кампании, контакты, команды и т.д.).

### Авторизация и идентификация

| Команда | Описание |
| :--- | :--- |
| `nudgen login` | Авторизация с помощью Personal Access Token (PAT). |
| `nudgen whoami` | Показать владельца текущего токена и информацию о пользователе. |
| `nudgen logout` | Удалить локальные учетные данные. |

### Управление командами (Teams)

| Команда | Описание |
| :--- | :--- |
| `nudgen teams list` | Список всех команд, в которых вы состоите. |
| `nudgen teams current` | Показать текущую активную команду. |
| `nudgen teams switch <id>` | Переключиться на конкретную команду. |

### Кампании (Campaigns)

Управляйте маркетинговыми усилиями в рамках активной команды.

| Команда | Описание |
| :--- | :--- |
| `nudgen campaigns list` | Список активных кампаний в текущей команде. |
| `nudgen campaigns create` | Интерактивный мастер создания кампании. |
| `nudgen campaigns update <id>` | Изменение настроек существующей кампании. |
| `nudgen campaigns delete <id>` | Удаление записи о кампании. |

### Контакты (Управление лидами)

| Команда | Описание |
| :--- | :--- |
| `nudgen contacts list` | Список всех лидов в текущей команде. |
| `nudgen contacts create` | Ручное добавление нового лида. |
| `nudgen contacts import` | Мастер импорта CSV для массовой загрузки. |
| `nudgen contacts update <id>` | Редактирование данных лида. |
| `nudgen contacts delete <id>` | Удаление записи о контакте. |

### Бренды (Brand)

| Команда | Описание |
| :--- | :--- |
| `nudgen brand list` | Список всех настроенных стилей (tone of voice) бренда. |
| `nudgen brand create` | Мастер создания нового стиля бренда. |
| `nudgen brand update <id>` | Редактирование конфигурации бренда. |
| `nudgen brand delete <id>` | Удаление стиля бренда. |

### Партнерство и доход

| Команда | Описание |
| :--- | :--- |
| `nudgen referral stats` | Сводная информация о результатах партнерства. |
| `nudgen referral link` | Получить вашу уникальную реферальную ссылку. |
| `nudgen referral activity` | История кликов и регистраций. |
| `nudgen referral referrals` | Список привлеченных команд и их статус. |
| `nudgen referral payouts` | История и статус запросов на выплату. |
