## Общая информация

Nudgen CLI — это мощный инструмент, разработанный как для людей, так и для ИИ-агентов. Он обеспечивает удобное управление кампаниями, контактами, брендами и партнерской статистикой прямо из терминала.

Независимо от того, являетесь ли вы разработчиком, желающим автоматизировать маркетинговые процессы, или ИИ-агентом, которому требуется качественное взаимодействие с платформой Nudgen, CLI — ваш основной интерфейс.

## Назначение

CLI выполняет несколько функций:

1. **Для людей**: Быстрый просмотр кампаний, импорт контактов и проверка доходов без необходимости открывать браузер.
2. **Для ИИ-агентов**: Предоставляет машиночитаемый интерфейс для автоматизации сложного маркетинга и управления брендами.
3. **Для разработчиков**: Помощь в локальной разработке и управлении репозиториями в экосистеме Nudgen.

## Модели взаимодействия

CLI следует принципу «Взаимодействие через API». Он использует стандартные запросы RESTful API для связи с бэкендом Nudgen, сохраняя состояние сессии локально в зашифрованном виде.

- **Стандарт JSON**: Все команды поддерживают флаг `--json`, что критически важно для взаимодействия агентов.
- **Учет контекста**: Такие операции, как вывод списка кампаний, автоматически ограничиваются вашей активной командой.
