Lead Email Finder совместим с **Google Chrome, Microsoft Edge**, всеми браузерами на базе Chromium и **Mozilla Firefox**.

## Поддерживаемые браузеры

Выберите ваш браузер ниже, чтобы перейти в официальный магазин дополнений:

  <Card title="Google Chrome" icon="chrome" href="https://chromewebstore.google.com/">
    Добавить в Chrome
  </Card>
  <Card title="Microsoft Edge" icon="edge" href="https://microsoftedge.microsoft.com/addons/">
    Добавить в Edge
  </Card>
  <Card title="Mozilla Firefox" icon="firefox-browser" href="https://addons.mozilla.org/">
    Добавить в Firefox
  </Card>
</CardGroup>

<Note>
Пользователи Brave, Opera и других браузеров на базе Chromium могут устанавливать расширение напрямую из Chrome Web Store.
</Note>

## Инструкция по настройке

1. **Перейдите в магазин**: Нажмите на ссылку выше для вашего браузера.
2. **Добавьте расширение**: Нажмите «Установить» (Add to Chrome/Edge) и подтвердите разрешения.
3. **Закрепите в тулбаре**: Для быстрого доступа нажмите на иконку пазла в браузере и закрепите **Lead Email Finder**.
4. **Начните поиск**: Откройте любой профиль или сайт и нажмите на иконку расширения, чтобы начать.

<Note>
Вы можете использовать расширение бесплатно с ежедневными лимитами сканирования. Для неограниченного использования перейдите на [Pro-план](https://nudgen.net/pricing).
</Note>
