## Что вы узнаете на этой странице

На этой странице вы узнаете, как настроить глобальную идентификацию вашего бренда. Эта информация используется в заголовках писем «От кого», предоставляет контекст ИИ для более точной генерации контента и устанавливает язык по умолчанию для текстов ваших кампаний.

## Перед началом работы

- Аккаунт Nudgen.
- Роль владельца или администратора.
- Активы вашего бренда (название, логотип и email отправителя).

## Настройки идентификации бренда

Перейдите в **Settings (Настройки)** → **Brand (Бренд)** на панели управления, чтобы получить доступ к этим параметрам:

| Опция                | Описание                                                                    |
| -------------------- | --------------------------------------------------------------------------- |
| **Brand name**       | Название вашей компании или продукта (например: «Nudgen»). Используется в контексте ИИ. |
| **Brand email**      | Email-адрес для ответов в ваших кампаниях по умолчанию (например: `support@yourbrand.com`). |
| **Brand description**| Краткое описание деятельности вашего бренда. Помогает ИИ писать тексты лучше. |
| **Email language**   | Язык, который ИИ использует по умолчанию при генерации тем и текстов писем. |
| **Logo**             | Логотип вашего бренда (PNG или JPEG, макс. 5 МБ). Используется в шаблонах писем и предпросмотрах. |
