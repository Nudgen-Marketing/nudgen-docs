## Invitations

**Invitee didn’t receive the invitation email**

- Ask them to check **spam/junk** and to whitelist or add the sender.
- Confirm the **email address** is correct. Resend the invitation from **Settings** → **Team** (use the resend action for the pending invite).
- If the invitation **expired**, create a new invitation for the same email. Expired links can’t be reused.

**I accepted an invite but I don’t see the workspace**

- Use the **team switcher** in the sidebar (header). Your new workspace should appear in the list. Select it to switch. See [Workspaces and roles](/en/concepts/workspaces-and-roles).

**Invitation link says expired or invalid**

- Links expire after a set period. Request a **new invitation** from the workspace owner or an admin. Don’t reuse old links.

## Roles and permissions

**I can’t invite members or change roles**

- Only **owners** and **admins** can invite members and change roles. **Members** can view the team list but cannot modify it. If you need admin access, ask an owner or admin to change your role. See [Team settings](/en/settings/team).

**I can’t access Billing or change the plan**

- **Billing** (upgrade, payment method, cancel) is restricted to **owners**. Admins and members can view usage but not manage the subscription. Ask the workspace owner to upgrade or update payment.

**A member was removed; can they get back in?**

- They need a **new invitation** from an owner or admin. Previous access is revoked when they’re removed.

## Workspace selection

**I see the wrong data / wrong campaigns or contacts**

- Data is **per workspace**. Ensure you’re in the correct workspace using the **team switcher** in the sidebar. Each workspace has its own contacts, campaigns, brand configs, and billing.

**I have multiple workspaces; how do I switch?**

- Click the current workspace name in the sidebar header to open the team switcher. Select the workspace you want. The page content updates to that workspace.

**I don’t see a workspace I was invited to**

- Confirm you **accepted** the invitation (clicked the link in the email). If you already accepted, log out and back in, or refresh and check the team switcher again. If it still doesn’t appear, the invitation may have been cancelled; ask for a new invite.
