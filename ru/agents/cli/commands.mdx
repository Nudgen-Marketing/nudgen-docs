## Обзор

Изучите основные команды, доступные в Nudgen CLI. Большинство команд сгруппированы по типу ресурса (кампании, контакты, команды и т. д.).

### Аутентификация и идентификация

| Команда | Описание |
| :--- | :--- |
| `nudgen login` | Аутентификация с помощью вашего персонального токена доступа (PAT). |
| `nudgen whoami` | Показать текущего владельца токена и информацию о пользователе. |
| `nudgen logout` | Удалить локальные учетные данные. |

### Управление командами

| Команда | Описание |
| :--- | :--- |
| `nudgen teams list` | Список всех команд, в которых вы состоите. |
| `nudgen teams current` | Показать текущий активный контекст команды. |
| `nudgen teams switch <id>` | Переключиться на конкретную команду. |

### Кампании

Управление маркетинговыми активностями в активной команде.

| Команда | Описание |
| :--- | :--- |
| `nudgen campaigns list` | Список активных кампаний в текущей команде. |
| `nudgen campaigns create` | Интерактивный мастер создания. |
| `nudgen campaigns update <id>` | Изменить настройки существующей кампании. |
| `nudgen campaigns delete <id>` | Удалить запись о кампании. |

### Контакты (Управление лидами)

| Команда | Описание |
| :--- | :--- |
| `nudgen contacts list` | Список всех лидов в текущей команде. |
| `nudgen contacts create` | Вручную добавить новую запись о лиде. |
| `nudgen contacts import` | Интерактивный CSV-маппер для массового импорта. |
| `nudgen contacts update <id>` | Редактировать данные лида. |
| `nudgen contacts delete <id>` | Удалить запись о контакте. |

### Идентичность бренда

| Команда | Описание |
| :--- | :--- |
| `nudgen brand list` | Список всех «голосов бренда» (brand voices) для команды. |
| `nudgen brand create` | Мастер инициализации нового голоса бренда. |
| `nudgen brand update <id>` | Редактировать конфигурации бренда. |
| `nudgen brand delete <id>` | Удалить идентичность бренда. |

### Рефералы и доходы

| Команда | Описание |
| :--- | :--- |
| `nudgen referral stats` | Просмотр краткого отчета об эффективности рефералов. |
| `nudgen referral link` | Получить вашу уникальную реферальную ссылку. |
| `nudgen referral activity` | Просмотр недавней истории кликов и регистраций. |
| `nudgen referral referrals` | Список команд, приглашенных вами, и их статус. |
| `nudgen referral payouts` | Просмотр истории запросов на выплату и их статуса. |

Далее узнайте, как использовать эти команды для [ИИ-агентов](/ru/agents/cli/agentic-usage).
