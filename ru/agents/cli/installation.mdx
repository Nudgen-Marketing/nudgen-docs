## Быстрая установка

 Самый простой способ установить Nudgen CLI — использовать наш скрипт установки в одну строку.

### macOS и Linux

Выполните следующую команду в терминале:

```bash
curl -fsSL https://raw.githubusercontent.com/Nudgen-Marketing/nudgen-cli/main/scripts/install.sh | bash
```

### Через Go

Если у вас установлен Go 1.22+, вы можете установить CLI напрямую:

```bash
go install github.com/Nudgen-Marketing/nudgen-cli@latest
```

### Из исходного кода

Если у вас установлен Go 1.22+, вы можете собрать проект из исходного кода:

```bash
git clone https://github.com/Nudgen-Marketing/nudgen-cli.git
cd nudgen-cli
make build
```

Бинарный файл будет доступен по пути `./bin/nudgen`.

## Первоначальная настройка

После установки проверьте работоспособность, узнав версию:

```bash
nudgen version
```

### Аутентификация

Чтобы начать использовать CLI, вам необходимо пройти аутентификацию с помощью персонального токена доступа (PAT).

1. Запустите команду входа:
   ```bash
   nudgen login
   ```
2. Следуйте интерактивным подсказкам, чтобы ввести свой токен.
3. Подтвердите свою личность:
   ```bash
   nudgen whoami
   ```

### Обновление

Чтобы обновиться до последней версии, просто снова запустите скрипт установки или используйте встроенную команду обновления (если она доступна):

```bash
nudgen update
```

## Безопасность

Nudgen CLI использует системную связку ключей (keychain) для безопасного хранения вашего PAT. Мы никогда не записываем и не выводим чистые токены ни в одном выводе команд.

Далее изучите [основные команды](/ru/agents/cli/commands).
