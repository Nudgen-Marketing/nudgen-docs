## Обзор

Nudgen CLI — это мощный инструмент, разработанный как для людей, так и для ИИ-агентов. Он обеспечивает удобный способ управления кампаниями, контактами, идентичностью бренда и статистикой рефералов прямо из вашего терминала.

Независимо от того, являетесь ли вы разработчиком, стремящимся автоматизировать свои маркетинговые рабочие процессы, или ИИ-агентом, которому требуется высококачественное взаимодействие с платформой Nudgen, CLI является вашим основным интерфейсом.

## Назначение

CLI выполняет несколько функций:

1. **Для людей**: обеспечивает быстрый способ просмотра кампаний, импорта контактов и проверки реферальных доходов без открытия веб-браузера.
2. **Для ИИ-агентов**: предлагает машиночитаемый интерфейс для автоматизации сложного маркетингового взаимодействия и управления брендом.
3. **Для разработчиков**: вспомогательные инструменты для локальной разработки и управления репозиториями в экосистеме Nudgen.

## Паттерны взаимодействия

CLI следует принципу «Взаимодействие через API». Он использует стандартные вызовы RESTful API для связи с бэкендом Nudgen, при этом безопасно управляя состоянием сессии локально.

- **Стандарт JSON**: все команды поддерживают флаг `--json`. Это критически важно для взаимодействия между агентами.
- **Учет контекста**: такие операции, как листинг кампаний, автоматически ограничиваются областью действия вашей активной команды.
