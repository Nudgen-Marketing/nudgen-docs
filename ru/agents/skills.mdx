## Добавьте возможностей вашему агенту

Навыки Nudgen — это модульные инструкции и скрипты, разработанные для агентов программирования, таких как Claude Code, Codex и Cursor. Они расширяют понимание вашим агентом маркетинговых рабочих процессов и паттернов интеграции Nudgen.

## Доступные навыки

### Навык CLI
Установка и аутентификация Nudgen CLI, управление командами, контактами, кампаниями, идентичностью бренда и отслеживанием рефералов.

```bash
npx skills add https://github.com/Nudgen-Marketing/skills --global --skill cli
```

**Подробности**: Установка, аутентификация через PAT и управление контактами, кампаниями, настройками бренда и реферальными данными из терминала. Поддерживает переключение контекста между несколькими командами и вывод в формате `--json` для скриптов.

---

### Навык API
CRUD контактов, кампании, команды, конфигурации бренда, статистика, данные партнеров, паттерны SDK, ограничения частоты запросов и ключи идемпотентности.

```bash
npx skills add https://github.com/Nudgen-Marketing/skills --global --skill api
```

**Подробности**: Охватывает контакты, кампании, шаблоны, настройки бренда, управление командами, паттерны SDK vs REST API, лимиты запросов, ключи идемпотентности и обработку ошибок. Использует `Authorization: Bearer <PAT>` для аутентификации.

---

### Навык «Лучшие практики Email»
Универсальный навык для работы с электронной почтой. Проводит аудит доставляемости, структуры и паттернов жизненного цикла. Работает как с Nudgen, так и без него.

```bash
npx skills add https://github.com/Nudgen-Marketing/skills --global --skill email-sending-best-practices
```

**Подробности**: Проверяет аутентификацию отправителя (SPF, DKIM, DMARC), контролирует разделение транзакционных и маркетинговых писем, анализирует обработку отписок, отмечает отсутствие этапов жизненного цикла и оценивает контент на наличие спам-триггеров.
