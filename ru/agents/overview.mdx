## Варианты использования

Как выглядит агентный маркетинг на практике? Вот несколько типичных рабочих процессов:

### «Протестируй это транзакционное письмо из терминала»
**Что делает агент:** Использует CLI для отправки тестового транзакционного письма с вашим ID шаблона и переменными с примерами данных. Показывает статус ответа и подтверждение доставки.

**Что завершаете вы:** Проверяете входящие сообщения, просматриваете отрендеренное письмо, публикуете шаблон, когда он будет готов.

---

### «Проведи аудит моей настройки почты и сопоставь её с Nudgen»
**Что делает агент:** Сканирует кодовую базу на предмет вызовов почтовых провайдеров, идентифицирует каждую отправку, шаблон и вебхук. Сопоставляет каждый элемент с эквивалентом в Nudgen и создает пофайловый план миграции.

**Что завершаете вы:** Воссоздаете шаблоны в редакторе Nudgen, настраиваете параметры отписки и вебхуков в панели управления.

---

### «Отправь событие отмены в Nudgen»
**Что делает агент:** Подключает ваш обработчик вебхуков Stripe для вызова API событий Nudgen при отмене подписки. Устанавливает имя события, сопоставляет соответствующие свойства (план, причина, дата).

**Что завершаете вы:** Создаете Nudgen (сценарий) отмены в панели управления, при необходимости определяете пользовательские свойства контактов.

---

### «Установи Nudgen в моё приложение Next.js»
**Что делает агент:** Обнаруживает Next.js, устанавливает JavaScript SDK, подключает NUDGEN_API_KEY из .env, создает маршрут синхронизации контактов и пример транзакционной отправки, выводит чеклист для тестирования.

**Что завершаете вы:** Добавляете API-ключ в панели управления Nudgen, публикуете шаблон транзакционного письма, проверяете тестовую отправку.

## Планы развития

Мы постоянно расширяем возможности наших агентов:

- **Навык настройки (Setup skill)**: (Скоро) Обнаруживает ваш стек, подключает авторизацию и создает рабочие примеры.
- **Навык миграции (Migrate skill)**: (Скоро) Проводит аудит вашего стека и строит план миграции.
- **Программное создание писем**: (Далее) Написание шаблонов писем с помощью кода.
