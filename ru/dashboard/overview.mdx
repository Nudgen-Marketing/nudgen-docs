## Что вы узнаете на этой странице

Вы познакомитесь с основным видом **панели управления (Dashboard)**: общими метриками для вашего рабочего пространства, графиком активности по времени и таблицей кампаний с ключевыми показателями. Используйте панель, чтобы мгновенно оценивать состояние и эффективность рассылок.

## Перед началом работы

- Аккаунт Nudgen хотя бы с одним рабочим пространством.
- Желательно: наличие отправленных или запланированных кампаний, чтобы на графиках и в метриках были данные.

## Карточки метрик

На панели управления отображаются итоговые показатели за выбранный период:

| Метка | Значение |
| ----- | -------- |
| **Emails Sent** | Общее количество всех попыток отправки (включая ошибки API или сети). |
| **Emails Delivered** | Количество писем, успешно принятых почтовым сервером получателя. |
| **Total Opens** | Сколько раз письма открывались (отслеживается пикселем). |
| **Total Clicks** | Сколько раз переходили по ссылкам. |
| **CTR** | Показатель кликабельности: (Всего кликов / Всего отправлено) × 100. |
| **Spam Complaints** | Количество раз, когда получатели помечали ваши письма как спам. |
| **Soft Bounce** | Временные проблемы с доставкой (ящик переполнен, ограничения, временные сбои сервера). |
| **Hard Bounce** | Критические ошибки доставки (неверные получатели, заблокированные доменные имена). |

Используйте эти данные для оценки доставляемости (доставлено vs отправлено, возвраты) и вовлеченности (открытия, клики, CTR). Большое количество жалоб на спам или возвратов может испортить репутацию отправителя.

## График эффективности

Линейный график показывает изменение метрик за выбранный диапазон дат. Вы можете менять диапазон с помощью календаря. График помогает выявлять тренды (например, падение открытий или всплеск возвратов) и сопоставлять их с запуском новых кампаний.

## Таблица кампаний

В таблице перечислены все кампании с колонками: название, тип (разовая или капельная), статус и счетчики (отправлено, доставлено, открыто, клики, возвраты, жалобы). Нажмите на кампанию, чтобы открыть детальную страницу с активностью по каждому получателю и специфическими графиками этой кампании.

## Диапазон дат

По умолчанию обычно отображаются последние 7 дней. Измените диапазон, чтобы сфокусироваться на конкретном периоде или сравнить разные недели. Статистика и график обновляются автоматически.
